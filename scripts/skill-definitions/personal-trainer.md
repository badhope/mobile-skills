## 📚 核心框架

### 训练原则

```
五大训练原则:
├── 渐进超负荷: 逐步增加重量/次数/组数/给肌肉新刺激
├── 特异性: 训练要针对目标/力量训练→力量/有氧→心肺
├── 个体化: 因人而异/考虑身体状况/训练经验
├── 恢复: 训练+休息=进步/肌肉在休息时生长
└── 周期化: 分阶段训练/准备期/强化期/恢复期

周期化训练:
├── 准备期(4-6周): 适应训练/建立基础
├── 强化期(6-8周): 增加强度/提升能力
├── 峰值期(2-4周): 达到最佳状态
└── 恢复期(1-2周): 主动休息/恢复调整
```

### 训练计划制定

```
计划制定流程:
评估 → 目标设定 → 训练设计 → 执行监控 → 调整优化

评估内容:
├── 身体评估: 体脂率/肌肉量/体态/运动能力
├── 健康评估: 伤病史/运动禁忌/健康状况
├── 目标评估: 减脂/增肌/塑形/体能提升
└── 时间评估: 每周训练频率/每次时长

训练计划要素:
├── 训练频率: 每周几次
├── 训练内容: 具体动作/组数/次数
├── 训练强度: 重量/速度/难度
├── 训练量: 总组数/总次数
└── 休息时间: 组间休息/训练间隔
```

### 力量训练

```
训练分化:
├── 全身训练: 每次训练全身/适合新手/每周3次
├── 上下肢分化: 上肢一天下肢一天/每周4次
├── 推拉腿分化: 推/拉/腿/每周3-6次
└── 单肌群分化: 每次一个肌群/每周5-6次

力量训练要点:
├── 复合动作优先: 深蹲/硬拉/卧推/推举/划船
├── 动作规范: 先掌握动作再增加重量
├── 渐进超负荷: 持续增加训练强度
├── 充分恢复: 同一肌群间隔48-72h
└── 训练记录: 记录重量/次数/感受

肌肥大训练参数:
├── 次数: 6-12次/组
├── 组数: 每个肌群10-20组/周
├── 强度: 65-85% 1RM
├── 休息: 60-90秒
└── 频率: 每个肌群2-3次/周
```

### 有氧训练

```
有氧训练类型:
├── 低强度稳态LISS: 30-60min/心率60-70% HRmax/适合减脂
├── 高强度间歇HIIT: 20-30min/心率80-90% HRmax/效率高
└── 中等强度: 30-45min/心率70-80% HRmax/平衡

心率区间:
├── 最大心率 = 220 - 年龄
├── 燃脂区间: 60-70% HRmax
├── 有氧区间: 70-80% HRmax
├── 无氧区间: 80-90% HRmax
└── 极限区间: 90-100% HRmax

有氧训练安排:
├── 减脂: 每周3-5次/每次30-45min
├── 增肌: 每周2-3次/每次20-30min
└── 体能提升: 每周4-6次/每次30-60min
```

### 运动康复

```
常见运动损伤:
├── 肌肉拉伤: 热身不足/动作过快
├── 关节扭伤: 动作不规范/意外
├── 肌腱炎: 过度使用/恢复不足
├── 腰背痛: 核心薄弱/姿势不良
└── 肩袖损伤: 动作错误/过度训练

损伤预防:
├── 充分热身: 5-10min动态热身
├── 动作规范: 先掌握技术再增加强度
├── 循序渐进: 不急于求成
├── 充分恢复: 不带伤训练
└── 营养补充: 蛋白质/胶原蛋白/维生素C

康复训练流程:
评估 → 急性期处理 → 恢复期训练 → 功能恢复 → 回归运动

康复训练原则:
├── 无痛原则: 训练不应引起疼痛
├── 渐进原则: 从简单到复杂
├── 个体化: 根据损伤情况调整
└── 专业指导: 严重损伤需医生指导
```

### 营养建议

```
训练营养:
├── 训练前: 碳水化合物为主/提前1-2小时
├── 训练中: 补充水分/长时间训练补充电解质
└── 训练后: 蛋白质+碳水化合物/30-60分钟内

蛋白质摄入:
├── 减脂期: 1.6-2.2g/kg体重
├── 增肌期: 2.0-2.5g/kg体重
├── 来源: 肉/蛋/奶/豆类/蛋白粉
└── 分配: 每餐20-40g/3-5餐

碳水化合物:
├── 减脂期: 2-3g/kg体重
├── 增肌期: 4-6g/kg体重
├── 来源: 米饭/面食/土豆/水果
└── 时机: 训练前后为主

补剂建议:
├── 蛋白粉: 方便补充蛋白质
├── 肌酸: 提升力量和爆发力
├── BCAA: 减少肌肉分解
└── 维生素D: 骨骼健康
```
