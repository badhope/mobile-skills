## 📚 核心框架

### 职业规划模型

```
职业规划五步法:
自我认知 → 职业探索 → 目标设定 → 行动计划 → 持续调整

自我认知维度:
├── 兴趣: 喜欢做什么/霍兰德测评
├── 能力: 擅长做什么/能力评估
├── 价值观: 想要什么/价值观澄清
└── 性格: 适合做什么/MBTI/DISC

职业探索维度:
├── 行业: 行业趋势/发展前景
├── 职业: 岗位要求/发展路径
├── 公司: 企业文化/发展机会
└── 机会: 市场需求/竞争分析
```

### 霍兰德职业兴趣

```
六种职业类型:
├── R现实型: 动手能力强/务实/工程/技术/操作
├── I研究型: 分析能力强/好奇/科研/分析/咨询
├── A艺术型: 创造力强/表达欲/设计/艺术/创意
├── S社会型: 人际能力强/助人/教育/服务/管理
├── E企业型: 领导力强/目标导向/管理/销售/创业
└── C常规型: 条理清晰/注重细节/财务/行政/数据

职业选择: 个人兴趣代码与职业环境代码匹配度越高越适合
```

### 目标设定

```
SMART目标原则:
├── Specific具体: 明确具体的目标
├── Measurable可衡量: 可量化的指标
├── Achievable可达成: 具有挑战但可实现
├── Relevant相关: 与职业愿景相关
└── Time-bound有时限: 明确的时间节点

职业目标层次:
├── 长期目标(5-10年): 职业愿景/最终目标
├── 中期目标(1-3年): 阶段性目标
└── 短期目标(3-6个月): 具体行动目标
```

### 能力提升

```
能力四象限:
├── 优势区: 高能力高兴趣 → 发挥优势/创造价值
├── 潜能区: 低能力高兴趣 → 投入学习/培养能力
├── 储备区: 高能力低兴趣 → 维持能力/需要时用
└── 盲区: 低能力低兴趣 → 委托他人/避免投入

能力提升路径:
├── 70%实践学习: 项目/任务/挑战
├── 20%社交学习: 导师/同伴/反馈
└── 10%正式学习: 培训/课程/阅读
```

### 职业转型

```
转型评估维度:
├── 动机: 为什么转型/内在驱动力
├── 能力: 现有能力/可迁移能力/需补充能力
├── 市场: 目标行业/职业需求/竞争情况
├── 风险: 经济压力/时间成本/机会成本
└── 支持: 家庭支持/人脉资源/学习资源

可迁移能力:
├── 通用能力: 沟通/协作/问题解决
├── 管理能力: 项目管理/团队管理
├── 专业技能: 数据分析/写作/设计
└── 软技能: 学习能力/适应能力/抗压能力

转型路径:
├── 平行转型: 同行业不同岗位
├── 跨行转型: 不同行业同岗位
├── 完全转型: 不同行业不同岗位
└── 创业转型: 从打工到创业
```

### 求职策略

```
求职渠道:
├── 内推: 成功率最高/利用人脉
├── 猎头: 中高端岗位/被动求职
├── 招聘网站: 海投/效率较低
├── 社交媒体: LinkedIn/行业社群
└── 校园招聘: 应届生/实习转正

简历优化要点:
├── 结构清晰: 个人信息/工作经历/教育背景/技能
├── 量化成果: 用数据说话/具体成果
├── 关键词匹配: 根据JD调整关键词
├── 突出亮点: 最相关的经历放前面
└── 篇幅控制: 1-2页/重点突出

面试准备:
├── 公司研究: 业务/文化/竞品/新闻
├── 自我介绍: 1-3分钟版本
├── 行为面试: STAR法则准备案例
├── 专业问题: 岗位相关专业知识
└── 提问环节: 准备3-5个有深度的问题
```
