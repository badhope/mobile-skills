## 📚 核心框架

### 导演艺术框架

```
导演职责模型:
├── 前期筹备: 剧本解读/创作方向/主创组建/演员选角/拍摄计划/美术风格
├── 拍摄制作: 现场调度/表演指导/镜头设计/部门协调/创作把控
└── 后期制作: 剪辑指导/声音设计/调色把控/特效审核/成片把控

视听语言:
├── 景别: 远景(环境)/全景(关系)/中景(动作)/近景(情绪)/特写(细节)
├── 运镜: 推(强调)/拉(退出)/摇(展示)/移(流动)/升降(视角)
├── 角度: 平视(客观)/俯视(压抑)/仰视(崇高)/倾斜(不安)
└── 光影: 方向/强度/色彩/氛围

场面调度:
├── 人物调度: 走位/动作/表情/互动
├── 空间调度: 场景/空间利用/景深/层次
├── 道具调度: 位置/使用/象征
└── 光影调度: 方向/强度/色彩/氛围
```

### 叙事结构

```
叙事结构类型:
├── 线性叙事: 时间顺序/因果关系 → 传统故事
├── 非线性叙事: 时间跳跃/多线并行 → 复杂故事
├── 环形叙事: 首尾呼应/循环结构 → 哲学主题
└── 碎片化叙事: 片段组合/留白艺术 → 艺术电影

叙事节奏:
├── 开场: 建立情境/吸引观众
├── 发展: 推进情节/深化冲突
├── 高潮: 冲突爆发/情感顶点
├── 结局: 解决冲突/情感释放
└── 节奏控制: 快慢结合/张弛有度

叙事技巧:
├── 伏笔: 提前埋下线索
├── 悬念: 保持观众好奇
├── 转折: 打破观众预期
└── 呼应: 前后呼应/形成闭环
```

### 表演指导

```
表演指导方法:
├── 情绪引导: 帮助演员进入情绪
├── 动作设计: 设计符合角色的动作
├── 台词处理: 指导台词的表达方式
├── 即兴创作: 允许演员即兴发挥
└── 多条拍摄: 尝试不同表演方式

演员沟通技巧:
├── 建立信任: 让演员信任导演
├── 清晰指令: 给出明确的表演方向
├── 情感支持: 给予演员情感支持
├── 尊重演员: 尊重演员的创作
└── 鼓励创新: 鼓励演员的创新
```

## 💼 实战模板

### 剧本解读模板

```
一、故事层面
├── 故事梗概: 一句话故事
├── 主题思想: 故事要表达什么
├── 核心冲突: 主要矛盾是什么
└── 情节结构: 开端/发展/高潮/结局

二、人物层面
├── 主要人物: 人物性格/动机/成长
├── 人物关系: 人物之间的关系
├── 人物弧光: 人物的成长变化
└── 人物功能: 人物在故事中的作用

三、风格层面
├── 类型定位: 影片类型
├── 影像风格: 视觉风格
├── 叙事风格: 叙事方式
└── 情感基调: 情感氛围

四、创作方向
├── 创作重点: 重点表现什么
├── 创新点: 有什么独特之处
├── 难点: 创作难点是什么
└── 解决方案: 如何解决难点
```

### 拍摄计划模板

```
一、前期筹备
├── 剧本定稿
├── 主创组建
├── 演员选角
├── 场景勘景
├── 美术设计
└── 拍摄计划

二、拍摄制作
├── 拍摄日程: 每日拍摄计划
├── 场次安排: 拍摄顺序
├── 部门协调: 各部门配合
└── 现场调度: 现场指挥

三、后期制作
├── 剪辑: 粗剪/精剪
├── 声音: 配音/音效/音乐
├── 调色: 色彩调整
├── 特效: 视觉特效
└── 成片: 最终输出
```

## 🚀 核心原则

- 故事为王: 好故事是好电影的基础
- 视听语言: 用视听语言讲故事
- 团队协作: 电影是集体创作
- 细节把控: 细节决定品质
- 艺术表达: 保持个人艺术风格
