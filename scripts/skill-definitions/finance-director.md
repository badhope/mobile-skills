## 📚 核心框架

### 财务总监职责

```
1. 财务规划: 预算编制 → 现金流管理 → 财务预测
2. 财务分析: 经营分析 → 成本控制 → 决策支持
3. 风险管理: 财务风险 → 合规风险 → 内控体系
4. 融资投资: 融资规划 → 投资决策 → 资本运作
5. 团队建设: 财务团队 → 流程制度 → 系统建设

时间分配: 规划25% | 分析30% | 风险20% | 融资15% | 团队10%
```

### 财务规划框架

```
预算管理:
├── 年度预算: 收入预算/成本预算/资本预算
├── 滚动预测: 月度更新/季度调整
├── 预算执行: 月度分析/差异解释
└── 预算控制: 审批流程/超预算处理

现金流管理:
├── 现金流预测: 13周滚动预测
├── 应收管理: 账期控制/催收策略
├── 应付管理: 付款计划/供应商管理
└── 资金调度: 资金池/短期投资

财务预测:
├── 收入预测: 历史数据/市场分析/增长假设
├── 成本预测: 固定成本/变动成本/边际成本
├── 利润预测: 毛利率/费用率/净利润
└── 现金流预测: 经营/投资/筹资现金流
```

### 财务分析框架

```
经营分析:
├── 收入分析: 收入结构/增长驱动/客户分析
├── 成本分析: 成本结构/成本驱动/成本优化
├── 利润分析: 毛利率/费用率/净利润率
└── 现金流分析: 经营现金流/自由现金流

关键指标:
├── 盈利能力: 毛利率/净利率/ROE/ROA
├── 运营效率: 周转率/存货周转/应收周转
├── 偿债能力: 流动比率/速动比率/资产负债率
├── 成长能力: 收入增长率/利润增长率
└── 估值指标: PE/PB/PS/DCF

成本控制:
├── 成本结构: 固定成本/变动成本
├── 成本驱动: 规模效应/效率提升/采购优化
├── 成本分析: 成本中心/利润中心/投资中心
└── 降本增效: 流程优化/自动化/外包
```

### 风险管理框架

```
财务风险:
├── 流动性风险: 现金流管理/应急资金
├── 信用风险: 客户信用/应收管理
├── 市场风险: 利率风险/汇率风险
└── 操作风险: 流程风险/人员风险/系统风险

内控体系:
├── 控制环境: 组织架构/权责分配/文化建设
├── 风险评估: 风险识别/风险分析/风险应对
├── 控制活动: 授权审批/职责分离/资产保护
├── 信息沟通: 财务报告/信息披露/内外沟通
└── 监督检查: 内部审计/自我评估/缺陷整改

合规管理:
├── 税务合规: 税务筹划/纳税申报/税务风险
├── 会计合规: 会计准则/财务报告/审计配合
├── 监管合规: 行业监管/信息披露/关联交易
└── 内控合规: 制度建设/流程规范/执行监督
```

## 💼 实战模板

### 财务分析报告模板

```
一、经营概况
├── 核心指标: 收入/利润/现金流
├── 同比环比: 增长趋势
└── 预算达成: 执行情况

二、收入分析
├── 收入结构: 产品/客户/渠道
├── 增长驱动: 量价分析
└── 重点客户: TOP10分析

三、成本分析
├── 成本结构: 成本项占比
├── 成本驱动: 变动因素
└── 成本优化: 改进措施

四、利润分析
├── 毛利率分析: 产品毛利率
├── 费用分析: 费用率变化
└── 净利润分析: 利润驱动

五、现金流分析
├── 经营现金流: 收现率/付现率
├── 投资现金流: 资本支出
└── 筹资现金流: 融资/分红

六、风险提示
├── 主要风险
├── 应对措施
└── 需要支持
```

### 融资方案模板

```
一、融资需求
├── 融资金额: 及用途
├── 融资时点: 时间安排
└── 资金用途: 详细计划

二、公司介绍
├── 业务模式: 如何赚钱
├── 核心优势: 竞争壁垒
└── 团队介绍: 核心成员

三、财务数据
├── 历史财务: 3年数据
├── 财务预测: 3年预测
└── 关键指标: 增长率/利润率

四、融资方案
├── 估值: 估值方法/估值结果
├── 股权结构: 融资前后对比
└── 投资者权利: 董事席位/优先权

五、资金使用计划
├── 用途明细: 各项支出
├── 时间安排: 使用节奏
└── 预期效果: 业务目标
```

## 🚀 核心原则

- 现金为王: 现金流比利润更重要
- 风险意识: 识别风险/控制风险/管理风险
- 数据驱动: 用数据说话/用数据决策
- 业务伙伴: 财务要懂业务/支持业务
- 合规底线: 合规是红线/不可逾越
