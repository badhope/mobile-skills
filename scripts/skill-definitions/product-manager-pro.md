## 📚 核心框架

### 产品战略框架

```
产品战略三层次:
├── 产品愿景: 我们要成为什么?长期目标是什么?
├── 产品战略: 如何实现愿景?聚焦哪些市场/用户?
└── 产品路线图: 具体要做什么?什么时候做?优先级如何?

产品矩阵管理:
├── 明星产品: 高增长/高份额 → 投资增长
├── 金牛产品: 低增长/高份额 → 维持利润
├── 问题产品: 高增长/低份额 → 决定投资或放弃
└── 瘦狗产品: 低增长/低份额 → 收割或退出

路线图时间框架:
├── 短期(0-3月): 明确功能清单/具体交付时间/资源已分配
├── 中期(3-6月): 方向性规划/季度目标/资源预估
└── 长期(6-12月): 战略方向/里程碑/粗略规划
```

### 产品团队管理

```
PM能力金字塔:
├── 初级PM(0-2年): 执行力/用户洞察/产品设计
├── 中级PM(2-5年): +数据分析/沟通能力/技术理解
├── 高级PM(5-8年): +商业sense/领导力
└── 产品总监(8年+): +战略思维

产品开发流程:
├── 发现阶段: 用户研究/问题定义/方案探索/假设验证
├── 定义阶段: 需求文档PRD/原型设计/技术评审/优先级
├── 开发阶段: Sprint规划/每日站会/进度追踪/风险管理
├── 发布阶段: 测试验收/发布准备/灰度发布/全量发布
└── 迭代阶段: 数据监控/用户反馈/效果评估/下一步规划

团队管理机制:
├── 周会: 进展同步/问题讨论/下周计划
├── 1on1: 工作进展/困难支持/职业发展
├── 月度复盘: 数据回顾/案例总结/经验沉淀
└── 季度规划: 目标制定/资源分配/风险评估
```

### 产品生命周期管理

```
引入期(0-1):
├── 特征: 用户少/产品不完善/商业模式未验证
├── 目标: 找到PMF(产品市场匹配)
├── 关键动作: 快速迭代/深度访谈/数据驱动/聚焦核心
└── 指标: 留存率/NPS/核心功能使用率

成长期(1-N):
├── 特征: 用户快速增长/产品成熟/竞争加剧
├── 目标: 规模化增长
├── 关键动作: 增长黑客/功能扩展/体验优化/品牌建设
└── 指标: DAU/MAU/CAC/LTV/市场份额

成熟期:
├── 特征: 增长放缓/用户基数大/竞争激烈/利润稳定
├── 目标: 维持份额/提升利润
├── 关键动作: 产品差异化/精细化运营/成本优化/新业务探索
└── 指标: 留存率/ARPU/利润率/用户满意度

衰退期:
├── 特征: 用户流失/收入下降/市场萎缩/技术过时
├── 目标: 收割价值/寻找转型
├── 关键动作: 成本削减/用户迁移/新产品孵化/退出决策
└── 指标: 现金流/流失率/新业务进展
```

### 商业化策略

```
商业模式类型:
├── 订阅制SaaS: 按月/年收费 → MRR/Churn/LTV
├── Freemium: 免费+付费 → 转化率/ARPPU
├── 广告: 免费使用/广告变现 → DAU/ARPU
├── 交易佣金: 抽取交易费用 → GMV/Take Rate
└── 增值服务: 基础免费/高级付费 → 付费率/ARPPU

定价策略:
├── 成本加成: 价格 = 成本 × (1 + 利润率)
├── 价值定价: 价格 = 用户感知价值
├── 竞争定价: 价格 = 竞品价格 ± 差异溢价
└── 动态定价: 价格 = 根据需求/供给实时调整

定价心理学:
├── 锚定效应: 先展示高价再展示实际价格
├── 奇数定价: ¥99比¥100感觉便宜很多
├── 套餐定价: 套餐比单品更划算
├── 免费试用: 降低决策门槛
└── 阶梯定价: 不同用户不同价格
```

## 💼 实战模板

### 产品战略规划模板

```
一、现状分析
├── 产品矩阵分析: 各产品营收/增长/市场份额/定位
├── 市场分析: 市场规模/增长趋势/竞争格局
└── 用户分析: 用户需求/痛点/行为特征

二、战略制定
├── 产品愿景: 3年愿景
├── 战略选择: 重点产品/维持投入/战略收缩
├── 差异化策略: 目标用户/核心价值/竞争优势
└── 资源分配: 各产品线资源投入比例

三、路线图规划
├── Q1: 重点功能/关键里程碑
├── Q2: 功能扩展/生态建设
├── Q3: 增值功能/新市场探索
└── Q4: 年度大版本/战略复盘
```

### 产品团队搭建模板

```
一、团队规划
├── 组织架构: 产品总监/高级PM/PM/运营/设计
├── 人员配置: 各角色人数/职责分工
└── 招聘优先级: 分批次招聘计划

二、招聘标准
├── 经验要求: 年限/行业/产品类型
├── 能力要求: 专业能力/软技能
├── 潜力评估: 学习能力/成长空间
└── 文化匹配: 价值观/工作方式

三、团队管理
├── 日常机制: 周会/1on1/月度复盘/季度规划
├── 绩效管理: OKR设定/月度check-in/季度评估
└── 团队建设: 培训/分享/团建/晋升
```
