## 📚 核心框架

### 曝光三要素

```
曝光三角:
光圈Aperture + 快门Shutter + 感光度ISO

光圈Aperture (f值):
├── 作用: 控制进光量 + 控制景深
├── f值越小 = 光圈越大 = 景深越浅(背景虚化)
├── f值越大 = 光圈越小 = 景深越深(前后清晰)
└── 常用参考:
    ├── f/1.4-f/2.8: 大光圈(人像/虚化背景/弱光)
    ├── f/4-f/5.6: 中等光圈(日常通用)
    ├── f/8-f/11: 小光圈(风景/前后清晰)
    └── f/16+: 最小光圈(星芒效果)

快门速度Shutter Speed:
├── 作用: 控制进光时间 + 决定格不定格运动
├── 速度越快 = 冻结动作
├── 速度越慢 = 运动模糊
└── 常用参考:
    ├── 1/4000s: 冻结高速运动(赛车/飞鸟)
    ├── 1/1000s: 体育/儿童/宠物
    ├── 1/125-1/250: 日常手持拍摄
    ├── 1/30s: 手持极限
    └── 1s-30s: 三脚架(流水/车轨/星空)

感光度ISO:
├── 作用: 控制传感器对光的敏感度
├── ISO越低 = 画质越细腻(噪点少)
├── ISO越高 = 画质越粗糙(噪点多)
└── 实用指南:
    ├── ISO 100-400: 最佳画质(光线充足)
    ├── ISO 800-1600: 可接受画质(室内/黄昏)
    └── ISO 3200+: 应急使用(明显噪点)
```

### 构图法则

```
经典构图法则:
├── 三分法Rule of Thirds:
│   └── 将画面分为九宫格/主体放在交叉点
├── 黄金分割Golden Ratio:
│   └── 1:1.618比例/更和谐
├── 对称构图Symmetry:
│   └── 左右或上下对称/正式感
├── 引导线Leading Lines:
│   └── 利用线条引导视线到主体
├── 框架构图Framing:
│   └── 用前景元素框住主体
└── 留白Negative Space:
    └── 主体周围留空/突出主体

视角选择:
├── 平视: 自然/真实
├── 俯视: 压迫/渺小
├── 仰视: 崇高/力量
└── 特殊视角: 航拍/微距/鱼眼
```

### 光线运用

```
光线类型:
├── 自然光:
│   ├── 黄金时段: 日出后/日落前1小时
│   ├── 蓝调时段: 日出前/日落后30分钟
│   └── 正午阳光: 硬光/对比强烈
├── 人造光:
│   ├── 持续光: LED/钨丝灯
│   └── 闪光灯: 瞬间强光/凝固动作
└── 混合光: 自然光+人造光

光位与效果:
├── 顺光Front Light: 平淡/清晰
├── 侧光Side Light: 立体感/质感
├── 逆光Back Light: 剪影/轮廓光
├── 顶光Top Light: 特殊效果/眼窝阴影
└── 底光Bottom Light: 恐怖/戏剧效果

光质:
├── 硬光Hard Light: 直射光/对比强/阴影清晰
└── 软光Soft Light: 漫射光/对比弱/阴影柔和
```
