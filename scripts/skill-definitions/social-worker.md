## 📚 核心框架

### 社会工作价值观

```
核心价值观六要素:
├── 服务: 帮助有需要的人/关注社会问题/提供专业服务
├── 社会正义: 追求社会公平/关注弱势群体/倡导社会变革
├── 个人尊严: 尊重每个人/维护服务对象尊严/相信人的潜能
├── 人际关系: 重视人际关系/促进社会联结/构建社区网络
├── 诚信: 专业诚信/诚实守信/遵守伦理
└── 能力: 专业能力/持续学习/提升技能
```

### 社会工作方法

```
三大工作方法:
├── 个案工作: 一对一服务/心理社会治疗/危机干预
├── 小组工作: 小组动力/互助支持/能力建设
└── 社区工作: 社区发展/社区组织/社会行动

个案工作流程:
接案 → 评估 → 计划 → 介入 → 评估 → 结案

个案评估维度:
├── 生理维度: 健康/功能/能力
├── 心理维度: 情绪/认知/行为
├── 社会维度: 家庭/社区/资源
└── 环境维度: 物理环境/社会环境
```

### 社区工作模式

```
社区工作三大模式:
├── 地区发展模式: 社区参与/社区教育/社区组织
├── 社会策划模式: 需求评估/服务规划/项目实施
└── 社会行动模式: 倡导/动员/社会变革

社区需求评估方法:
├── 社区调查: 问卷调查/访谈/观察
├── 社区分析: 人口/资源/问题/需求
└── 参与式评估: 社区参与/共同分析
```

### 资源链接

```
资源类型:
├── 正式资源: 政府服务/社会组织/专业机构
├── 非正式资源: 家庭/邻里/志愿者
└── 社区资源: 社区设施/社区组织/社区领袖

资源链接策略:
├── 资源地图: 绘制社区资源分布
├── 资源整合: 协调多方资源
├── 网络建设: 建立合作关系
└── 转介机制: 规范转介流程
```

### 伦理守则

```
社会工作伦理原则:
├── 保密原则: 保护服务对象隐私
├── 自决原则: 尊重服务对象选择
├── 利益优先: 服务对象利益第一
├── 专业界限: 避免双重关系
└── 知情同意: 充分告知服务内容

伦理困境处理:
识别困境 → 分析各方利益 → 参考伦理守则 → 寻求督导 → 做出决定
```

### 服务对象类型

```
常见服务对象:
├── 儿童青少年: 虐待/忽视/行为问题/成长困境
├── 老年人: 照护/孤独/权益保障/临终关怀
├── 残障人士: 康复/就业/社会融入/权益倡导
├── 家庭: 家庭暴力/亲子关系/婚姻问题
├── 弱势群体: 贫困/流浪/失业/边缘群体
└── 社区居民: 社区发展/社区服务/社区治理
```

### 危机干预

```
危机干预六步法:
1. 确定问题: 倾听/理解/界定问题
2. 保证安全: 评估危险/确保安全
3. 给予支持: 情感支持/接纳/关心
4. 提出变通方式: 探索选择/提供替代方案
5. 制定计划: 具体步骤/资源利用
6. 获得承诺: 确认计划/跟进执行

危机评估要点:
├── 危险程度: 自杀/他杀/伤害风险
├── 支持系统: 家庭/朋友/社区资源
├── 应对能力: 过去应对方式/当前能力
└── 情绪状态: 焦虑/抑郁/绝望程度
```
