## 📚 核心框架

### HR战略框架

```
HR三支柱模型:
├── HRBP(业务伙伴): 贴近业务/提供解决方案/支持业务目标
├── COE(专家中心): 制定政策标准/提供专业咨询/设计方案
└── SSC(共享服务中心): 标准化事务/提升效率/员工自助

人力资源规划:
├── 需求预测: 业务目标 ÷ 人均产出
├── 供给分析: 内部盘点/外部市场
└── 平衡策略: 招聘/培训/外包/优化

组织架构类型:
├── 职能型: 按专业分工 → 专业深入/协作困难
├── 事业部制: 按业务线分 → 灵活响应/资源分散
├── 矩阵式: 双重汇报 → 专业协同/权责不清
└── 扁平化: 层级少 → 决策快/管理幅度大
```

### 人才管理体系

```
招聘流程:
需求分析 → 渠道选择 → 简历筛选 → 面试评估 → 背景调查 → 录用决策 → 入职管理

招聘渠道:
├── 内部推荐: 质量高/成本低
├── 招聘网站: 覆盖广
├── 猎头: 高端岗位
├── 校园招聘: 应届生
└── 社交媒体: 品牌曝光

绩效管理方法:
├── KPI: 目标明确/量化指标/战略挂钩
├── OKR: 创新工作/挑战目标/透明公开
├── 360度评估: 多维反馈/管理岗位/发展导向
└── BSC: 财务/客户/流程/学习四维度

绩效管理闭环:
目标设定 → 过程辅导 → 绩效评估 → 反馈改进 → 结果应用

薪酬结构:
基本工资 + 绩效奖金 + 长期激励 + 福利

薪酬设计原则:
├── 内部公平: 岗位价值评估
├── 外部竞争: 市场薪酬调研
├── 个人公平: 能力与绩效挂钩
└── 企业承受: 成本可控

培训体系:
新员工培训 → 专业技能培训 → 管理能力培训 → 领导力发展

培训效果评估(柯氏四级):
├── 反应层: 学员满意度
├── 学习层: 知识技能掌握
├── 行为层: 工作行为改变
└── 结果层: 业务绩效提升

人才发展通道:
├── 管理通道: 员工→主管→经理→总监→VP
├── 专业通道: 初级→中级→高级→专家→首席
└── 双通道: 管理与专业并行
```

### 组织发展与文化建设

```
组织诊断模型:
├── 韦斯伯德六盒: 使命/结构/关系/激励/领导/支持
├── 麦肯锡7S: 战略/结构/制度/价值观/技能/人员/风格
└── 组织健康度: 方向/执行/革新/活力

企业文化层次:
├── 表层文化(物质层): 环境/标识/着装/仪式
├── 中层文化(行为层): 行为规范/工作方式/沟通风格
├── 深层文化(制度层): 规章制度/流程/激励机制
└── 核心文化(精神层): 价值观/使命/愿景

文化建设步骤:
诊断现状 → 提炼文化 → 宣导传播 → 落地践行 → 评估效果

文化落地方式:
├── 领导示范(最重要)
├── 制度保障
├── 典型案例
├── 仪式活动
├── 环境营造
└── 故事传播
```

## 💼 实战模板

### 组织变革模板

```
一、变革诊断
├── 现状分析: 组织架构/主要问题/变革目标
├── 阻力分析: 中层/员工/习惯/能力
└── 变革准备: 高管共识/变革团队/资源准备

二、变革方案
├── 组织架构调整: 新架构/人员配置/职责划分
├── 人员调整: 选拔/招聘/转岗/优化
└── 配套机制: 流程/制度/系统

三、变革实施
├── 准备阶段: 宣导/共识/人选确定
├── 试点阶段: 试点运行/问题收集/方案调整
├── 推广阶段: 全面推广/全员培训/新流程上线
└── 稳定阶段: 问题解决/效果评估/持续优化
```

### 绩效体系优化模板

```
一、现状诊断
├── 问题分析: 指标/周期/反馈/应用/参与度
├── 员工反馈: 满意度/改进建议
└── 管理者反馈: 执行难点/支持需求

二、体系设计
├── 考核方式: 研发OKR/销售KPI/职能KPI+行为/管理KPI+360
├── 考核流程: 目标设定→过程跟进→期中评估→期末评估
└── 结果应用: 奖金/调薪/晋升/培训/PIP

三、实施要点
├── 培训先行: 管理者/员工/HR培训
├── 试点运行: 试点部门/问题收集/优化方案
└── 持续优化: 季度回顾/问题收集/方案调整
```

## 🚀 核心原则

- 懂业务: 理解业务战略/痛点/语言
- 系统思维: 人的问题往往是组织的问题
- 数据驱动: 用数据验证效果/支撑决策
- 变革管理: 沟通比方案更重要
- 业务伙伴: 平等对话/专业判断/敢于说不
