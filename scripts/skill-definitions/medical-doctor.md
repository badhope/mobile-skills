## 📚 核心框架

### 红旗症状(需立即就医)

```
紧急情况(拨打120):
├── 胸痛/胸闷: 压榨性疼痛放射到左肩手臂 → 心肌梗死
├── 剧烈头痛: "一生中最痛的头痛" → 脑出血/动脉瘤破裂
├── 呼吸困难: 窒息感 → 肺栓塞/哮喘/心力衰竭
├── 中风FAST: 面部不对称/肢体无力/言语不清 → 脑卒中
├── 高热+意识改变: >39°C+颈部僵硬 → 脑膜炎
├── 大量呕血/黑便 → 消化道大出血
└── 严重过敏: 荨麻疹+呼吸困难+血压下降 → 过敏性休克
```

### 常见症状判断

```
可观察症状:
├── 普通感冒: 7-10天自限性/>10天或高热>3天就医
├── 轻微腹泻: 1-3天/>3天或脱水或便血就医
├── 轻微头痛: 数小时-1天/突然加重或伴发热呕吐就医
├── 轻微皮疹: 1-2天/迅速扩散或伴发热就医
└── 腰背酸痛: 数天-1周/>2周或伴麻木或大小便异常就医

就医时机判断:
├── 紧急: 红旗症状 → 立即就医/拨打120
├── 急诊: 症状严重/持续加重 → 24小时内就医
├── 门诊: 症状明显/影响生活 → 预约门诊
└── 观察: 症状轻微/无红旗症状 → 观察1-3天
```

### 体检报告解读

```
血常规核心指标:
├── WBC白细胞(3.5-9.5×10⁹/L)
│   ├── 偏高: 感染/炎症/白血病
│   └── 偏低: 免疫低下/病毒感染
├── RBC红细胞(男4.0-5.5/女3.5-5.0×10¹²/L)
│   ├── 偏高: 脱水/肺部疾病
│   └── 偏低: 贫血/出血/营养不良
├── HGB血红蛋白(男120-160/女110-150g/L)
│   └── 偏低: 贫血(最常见)
└── PLT血小板(125-350×10⁹/L)
    ├── 偏高: 原发性血小板增多症
    └── 偏低: ITP/再障/药物影响

代谢指标:
├── 血糖GLU: 空腹3.9-6.1mmol/L/餐后2h<7.8
├── 糖化血红蛋白HbA1c: <6.0%(反映近3月平均血糖)
├── 总胆固醇TC: <5.2mmol/L理想/5.2-6.2边缘/>6.2升高
├── LDL低密度脂蛋白: <3.4mmol/L(坏胆固醇)
├── HDL高密度脂蛋白: >1.0mmol/L(好胆固醇)
└── 甘油三酯TG: <1.7mmol/L

肝功能指标:
├── ALT谷丙转氨酶: <40U/L/升高提示肝损伤
├── AST谷草转氨酶: <40U/L/升高提示肝/心肌损伤
└── 胆红素: 总胆红素<21μmol/L

肾功能指标:
├── 肌酐Cr: 男53-106/女44-97μmol/L
├── 尿素氮BUN: 2.9-8.2mmol/L
└── 尿酸UA: 男150-416/女89-357μmol/L
```

### 常见疾病

```
高血压:
├── 诊断: 收缩压≥140/舒张压≥90mmHg
├── 分级: 1级(140-159/90-99)/2级(160-179/100-109)/3级(≥180/≥110)
├── 管理: 生活方式干预+药物治疗
└── 目标: <140/90mmHg(糖尿病/肾病<130/80)

糖尿病:
├── 诊断: 空腹血糖≥7.0mmol/L/餐后2h≥11.1/HbA1c≥6.5%
├── 分型: 1型(胰岛素依赖)/2型(最常见)
├── 管理: 饮食控制+运动+药物/胰岛素
└── 目标: 空腹4.4-7.0/餐后<10.0/HbA1c<7.0%

高血脂:
├── 诊断: TC≥6.2/LDL≥4.1/TG≥2.3mmol/L
├── 危害: 动脉粥样硬化/心脑血管疾病
├── 管理: 饮食控制+运动+他汀类药物
└── 目标: LDL<2.6(高危<1.8)

脂肪肝:
├── 诊断: B超/CT发现肝脏脂肪浸润
├── 类型: 酒精性/非酒精性
├── 管理: 减重/戒酒/控制血糖血脂
└── 预后: 可逆转/不控制可发展为肝硬化
```

### 用药安全

```
用药原则:
├── 遵医嘱: 按时按量服药/不自行调整剂量
├── 了解药物: 知道药物作用/副作用/禁忌
├── 注意相互作用: 多种药物同时使用需咨询医生
├── 定期复查: 监测药物效果和副作用
└── 不随意停药: 特别是慢性病药物

常见药物相互作用:
├── 阿司匹林+华法林: 出血风险增加
├── 他汀类+红霉素: 肌肉损伤风险增加
├── 降压药+西柚汁: 降压效果增强
├── 抗生素+酒精: 双硫仑样反应(头孢类)
└── 降糖药+酒精: 低血糖风险

特殊人群用药:
├── 孕妇: 避免致畸药物/咨询医生
├── 儿童: 儿童专用剂型/按体重计算剂量
├── 老年人: 从小剂量开始/注意肝肾功能
└── 肝肾功能不全: 调整剂量/选择安全药物
```

### 预防保健

```
疫苗接种:
├── 儿童: 按国家免疫规划接种
├── 成人: 流感疫苗(每年)/肺炎疫苗(65岁以上)
├── 特殊人群: 乙肝疫苗/HPV疫苗
└── 旅行: 根据目的地接种相关疫苗

筛查建议:
├── 血压: 18岁以上每年至少1次
├── 血糖: 40岁以上每年1次/高危人群更频繁
├── 血脂: 40岁以下每5年/40岁以上每年1次
├── 肿瘤筛查: 乳腺癌(40岁以上女性)/宫颈癌(25岁以上女性)/结直肠癌(50岁以上)
└── 骨密度: 65岁以上女性/70岁以上男性

健康生活方式:
├── 饮食: 均衡饮食/少盐少油少糖/多吃蔬果
├── 运动: 每周150分钟中等强度有氧运动
├── 睡眠: 7-9小时/规律作息
├── 戒烟限酒: 完全戒烟/限制饮酒
└── 心理: 保持积极心态/管理压力
```
