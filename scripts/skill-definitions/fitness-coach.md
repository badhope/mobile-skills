## 📚 核心框架

### 训练基础原理

```
FITT原则:
├── Frequency频率: 新手3-4次/周/进阶4-5次/高级5-6次
├── Intensity强度: 力量训练65-85% 1RM/有氧60-80% HRmax
├── Time时间: 力量45-75min/有氧20-60min
└── Type类型: 抗阻/有氧/柔韧性/功能性

渐进超负荷方法:
├── 增加重量(最直接)
├── 增加次数(同一重量做更多)
├── 增加组数(更多训练量)
├── 缩短组间休息(增加密度)
├── 增加训练频率(同一肌群更频繁刺激)
└── 改善动作控制(更好的神经募集)

恢复原则:
├── 同一肌群间隔48-72h再训练
├── 睡眠7-9h是最佳恢复手段
├── 蛋白质1.6-2.2g/kg体重
└── 主动恢复(轻度活动促进血液循环)
```

### 减脂科学

```
热量缺口:
每日消耗 > 每日摄入 = 减脂
每缺口7700kcal ≈ 减1kg脂肪
建议每日缺口300-500kcal(每周减0.3-0.7kg)

NEAT非运动性热消耗:
├── 占总消耗15-30%(比运动更容易控制)
├── 多走路(目标8000-10000步/天)
├── 站立办公/走楼梯不坐电梯
└── 做家务也算

HIIT vs LISS:
├── HIIT高强度间歇(20-30min): 效率高/EPOC后燃效应/但对新手关节压力大/每周2-3次
└── LISS低强度稳态(30-60min): 更温和/适合大基数人群/可天天做/不易过度训练

减脂期力量训练不可少:
├── 保住肌肉(否则减掉的不仅是脂肪)
├── 维持代谢率(肌肉=高代谢组织)
└── 塑造体型(瘦了但松垮不好看)
```

### 增肌训练

```
营养需求:
├── 盈余热量: 每日盈余200-300kcal(干净增肌)
├── 蛋白质: 2-2.2g/kg体重
├── 碳水化合物: 4-6g/kg(训练燃料)
└── 脂肪: 0.8-1g/kg(激素健康)

训练容量:
├── 每周每个肌群10-20组有效组数(新手取下限老手上限)
├── 每组6-12次肌肥大范围
└── 组间休息90s-3min(复合动作长隔离短)

复合动作优先:
深蹲/硬拉/卧推/推举/划船/引体向上
→ 这些动作同时调动多关节多肌群效率最高
→ 先做复合再做孤立
```

### 体态矫正

```
常见体态问题:
├── 圆肩驼背: 胸肌紧张/背部肌群薄弱
├── 头前伸: 颈部肌肉紧张/深层颈屈肌薄弱
├── 骨盆前倾: 髂腰肌紧张/臀肌薄弱
└── 腰椎过伸: 核心薄弱/髋屈肌紧张

矫正流程:
评估 → 松解紧张肌群 → 激活薄弱肌群 → 强化正确模式 → 日常习惯建立

圆肩驼背矫正:
├── 松解: 胸肌拉伸/胸椎伸展
├── 激活: 菱形肌/中下斜方肌
├── 强化: 划船/面拉/反向飞鸟
└── 日常: 每小时做墙臂扩展30秒

骨盆前倾矫正:
├── 松解: 髂腰肌拉伸/股直肌拉伸
├── 激活: 臀桥/臀推
├── 强化: 核心训练/臀部训练
└── 日常: 避免久坐/每小时起立活动
```

### 训练计划模板

```
减脂训练计划(每周4次):
├── Day1: 上肢推(胸/肩/三头) + HIIT 20min
├── Day2: 下肢(腿/臀) + 核心
├── Day3: 休息或LISS 30min
├── Day4: 上肢拉(背/二头) + HIIT 20min
├── Day5: 下肢(腿/臀) + 核心
├── Day6: 休息或LISS 30min
└── Day7: 休息

增肌训练计划(每周5次):
├── Day1: 胸+三头
├── Day2: 背+二头
├── Day3: 休息
├── Day4: 腿+臀
├── Day5: 肩+核心
├── Day6: 手臂或弱项训练
└── Day7: 休息

居家训练(无器械):
├── 上肢: 俯卧撑/倒立撑/臂屈伸
├── 下肢: 深蹲/箭步蹲/臀桥/提踵
├── 核心: 平板支撑/死虫式/卷腹
└── 有氧: 开合跳/波比跳/高抬腿
```

### 损伤预防

```
常见运动损伤:
├── 肩袖损伤: 动作不规范/过度训练
├── 腰椎间盘突出: 硬拉姿势错误/核心薄弱
├── 膝关节损伤: 深蹲膝盖内扣/过度训练
└── 肌肉拉伤: 热身不足/动作过快

预防措施:
├── 充分热身: 5-10分钟动态热身
├── 动作规范: 先掌握动作再增加重量
├── 循序渐进: 不急于求成
├── 充分恢复: 不带伤训练
└── 营养补充: 蛋白质/胶原蛋白/维生素C

损伤处理RICE原则:
├── Rest休息: 停止运动
├── Ice冰敷: 24-48小时内冰敷
├── Compression压迫: 弹性绷带包扎
└── Elevation抬高: 抬高患肢
```
