## 📚 核心框架

### 人生规划四维度

```
人生四维度平衡:
├── 事业Career: 职业发展/事业成就/财务目标
├── 关系Relationship: 家庭/伴侣/朋友/社交
├── 健康Health: 身体健康/心理健康/生活方式
└── 成长Growth: 学习成长/兴趣爱好/精神追求

平衡评估: 每个维度1-10分评分 → 识别失衡领域 → 制定改善计划
```

### 生活规划流程

```
规划五步法:
现状评估 → 愿景描绘 → 目标设定 → 行动计划 → 持续调整

现状评估:
├── 四维度现状盘点
├── 满意度评分(1-10分)
├── 优势与挑战识别
└── 核心需求明确

愿景描绘:
├── 理想生活画面
├── 价值观澄清
├── 人生使命探索
└── 长期愿景设定
```

### 目标设定系统

```
目标层次:
├── 愿景目标: 5-10年/人生方向
├── 战略目标: 1-3年/阶段性目标
├── 战术目标: 3-6个月/具体目标
└── 行动目标: 每周每日/具体行动

OKR目标管理:
Objective: 目标(定性/激励人心)
├── KR1: 关键结果(定量/可衡量)
├── KR2: 关键结果(定量/可衡量)
└── KR3: 关键结果(定量/可衡量)

目标达成公式:
明确目标 + 具体计划 + 持续行动 + 定期复盘 = 目标达成
```

### 习惯养成系统

```
习惯回路模型:
提示Cue → 行为Routine → 奖励Reward

习惯养成策略:
├── 微习惯: 从极小的行动开始(如每天1个俯卧撑)
├── 习惯叠加: 新习惯叠加在现有习惯上
├── 环境设计: 让好习惯更容易/坏习惯更难
├── 即时奖励: 给自己即时正向反馈
└── 社交支持: 找伙伴/公开承诺

习惯养成时间:
├── 简单习惯: 21天
├── 中等习惯: 66天
└── 复杂习惯: 90天以上

习惯追踪:
├── 日历打钩法
├── 习惯追踪APP
├── 习惯打卡群
└── 周复盘总结
```

### 时间管理

```
时间管理矩阵(艾森豪威尔矩阵):
├── 重要紧急: 立即处理/危机/截止日期任务
├── 重要不紧急: 计划安排/学习/健康/关系 ← 重点
├── 不重要紧急: 委托他人/部分会议/部分邮件
└── 不重要不紧急: 尽量不做/无意义刷手机

时间管理技巧:
├── 时间块: 固定时间做固定事情
├── 番茄工作法: 25分钟专注+5分钟休息
├── 二八法则: 20%重要任务创造80%价值
├── 批量处理: 同类任务集中处理
└── 缓冲时间: 预留20%缓冲时间

每日三件事:
每天确定最重要的3件事 → 优先完成 → 其他时间处理其他事务
```

### 精力管理

```
精力四维度:
├── 体能精力: 睡眠/运动/饮食/休息
├── 情感精力: 积极情绪/人际关系/自我接纳
├── 思维精力: 专注力/创造力/乐观思维
└── 意志精力: 价值观/使命感/热情

精力恢复策略:
├── 休息: 小憩/冥想/散步
├── 运动: 有氧运动/力量训练
├── 睡眠: 7-9小时/规律作息
├── 营养: 均衡饮食/充足饮水
└── 社交: 与正能量的人在一起

精力高峰期利用:
├── 识别自己的精力高峰时段
├── 高峰期做重要困难任务
└── 低谷期做简单机械任务
```

### 工作生活平衡

```
边界设定:
├── 时间边界: 工作时间/家庭时间/个人时间
├── 空间边界: 工作区/生活区分离
├── 心理边界: 不把工作情绪带回家
└── 数字边界: 下班后不回工作消息

平衡策略:
├── 明确优先级: 家庭/健康/工作的顺序
├── 学会说"不": 拒绝不合理要求
├── 全情投入: 工作时工作/陪伴时陪伴
├── 定期休假: 强制休息/充电
└── 寻求支持: 家人/朋友/专业人士
```
