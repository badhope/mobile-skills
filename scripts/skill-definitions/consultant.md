## 📚 核心框架

### 咨询流程

```
咨询项目全流程:
项目启动 → 诊断分析 → 方案设计 → 实施支持 → 项目收尾

项目启动:
├── 需求理解: 明确客户期望
├── 范围界定: 确定项目边界
├── 团队组建: 组建项目团队
└── 计划制定: 时间/资源/里程碑

诊断分析:
├── 现状调研: 访谈/问卷/数据分析
├── 问题识别: 发现关键问题
├── 根因分析: 找到问题根源
└── 假设验证: 验证分析结论
```

### 问题解决方法

```
麦肯锡问题解决方法:
定义问题 → 结构化分析 → 优先级排序 → 工作计划 → 分析验证 → 综合建议

定义问题:
├── 明确问题陈述
├── 确定问题边界
├── 识别利益相关者
└── 设定成功标准

MECE原则:
├── Mutually Exclusive: 各部分相互独立
├── Collectively Exhaustive: 各部分完全穷尽
└── 应用: 问题分解/方案设计

假设驱动:
├── 提出假设: 基于经验和直觉
├── 验证假设: 数据和分析
├── 修正假设: 根据验证结果
└── 形成结论: 基于证据
```

### 分析工具

```
战略分析工具:
├── PESTEL: 政治/经济/社会/技术/环境/法律
├── 五力模型: 供应商/客户/替代品/新进入者/竞争者
├── SWOT: 优势/劣势/机会/威胁
├── 价值链: 主要活动/支持活动
└── BCG矩阵: 明星/金牛/问题/瘦狗

运营分析工具:
├── 流程图: 流程可视化
├── 鱼骨图: 根因分析
├── 帕累托图: 识别关键少数
├── 控制图: 过程监控
└── 散点图: 相关性分析

财务分析工具:
├── 杜邦分析: ROE分解
├── 财务比率: 流动性/盈利性/效率
├── 现金流分析: 经营/投资/筹资
└── 盈亏平衡: 固定成本/变动成本
```

### 战略规划

```
战略三层次:
├── 公司层战略: 业务组合/资源配置
├── 业务层战略: 竞争战略/市场定位
└── 职能层战略: 营销/运营/人力资源

战略制定流程:
战略分析 → 战略选择 → 战略实施

战略分析:
├── 外部环境: PESTEL/五力模型
├── 内部环境: 资源/能力/核心竞争力
└── SWOT分析: 匹配内外部因素

竞争战略:
├── 成本领先: 低成本优势
├── 差异化: 独特价值主张
└── 聚焦: 细分市场专注

战略实施:
├── 组织结构调整
├── 资源配置优化
├── 流程再造
├── 文化变革
└── 绩效管理
```

### 变革管理

```
变革类型:
├── 结构变革: 组织架构调整
├── 流程变革: 业务流程优化
├── 技术变革: 数字化转型
└── 文化变革: 价值观/行为改变

变革管理模型:
├── Lewin三阶段: 解冻→变革→再冻结
├── Kotter八步法: 建立紧迫感→组建团队→制定愿景→沟通愿景→授权行动→创造短期胜利→巩固成果→深植文化
└── ADKAR: 意识→愿望→知识→能力→巩固

变革阻力:
├── 个人层面: 习惯/恐惧/利益受损
├── 组织层面: 结构惯性/群体惯性/权力威胁
└── 应对策略: 沟通/参与/支持/谈判/强制

变革成功要素:
├── 高层支持: 领导者承诺和参与
├── 清晰愿景: 明确变革方向
├── 有效沟通: 持续沟通变革信息
├── 员工参与: 让员工参与变革
└── 快速胜利: 创造短期可见成果
```

### 方案设计

```
方案设计原则:
├── 问题导向: 针对核心问题
├── 可执行性: 方案可落地
├── 成本效益: 投入产出合理
├── 风险可控: 风险可识别和管理
└── 可衡量性: 效果可评估

方案结构:
├── 背景: 为什么需要这个方案
├── 目标: 方案要达成什么
├── 现状: 当前情况分析
├── 方案: 具体解决方案
├── 实施: 如何执行
├── 风险: 可能的风险和应对
└── 效果: 如何衡量成功

实施计划:
├── 时间计划: 里程碑/时间节点
├── 资源计划: 人员/预算/设备
├── 风险计划: 风险识别和应对
└── 沟通计划: 利益相关者沟通
```

### 沟通与呈现

```
金字塔原理:
├── 结论先行: 核心观点在最前面
├── 以上统下: 上层观点统领下层
├── 归类分组: 同类思想归为一组
└── 逻辑递进: 按逻辑顺序组织

咨询报告结构:
├── 执行摘要: 核心发现和建议
├── 项目背景: 为什么做这个项目
├── 分析方法: 如何进行分析
├── 主要发现: 分析结果
├── 建议: 基于发现的建议
└── 附录: 详细数据和分析

演示技巧:
├── 开场: 吸引注意力
├── 结构: 逻辑清晰
├── 可视化: 图表辅助理解
├── 互动: 与听众互动
└── 总结: 强调核心信息
```
