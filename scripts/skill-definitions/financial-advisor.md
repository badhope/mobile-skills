## 📚 核心框架

### 理财规划流程

```
理财五步法:
财务诊断 → 目标设定 → 策略制定 → 执行实施 → 持续优化

财务健康评估:
├── 收支分析: 收入结构/支出结构/储蓄率
├── 资产负债: 资产结构/负债结构/净资产
├── 现金流: 流动性/应急储备
└── 风险敞口: 保险覆盖/风险承受能力

储蓄率标准:
├── <10%: 需改善
├── 10-20%: 基础
├── 20-30%: 良好
├── 30-50%: 优秀
└── >50%: 财务自由加速
```

### 资产配置

```
资产类别:
├── 现金类: 活期/货币基金/短期理财 → 流动性/应急
├── 固收类: 债券/债券基金/定期存款 → 稳定收益
├── 权益类: 股票/股票基金/指数基金 → 长期增长
├── 另类资产: 房产/黄金/REITs/私募 → 分散风险
└── 保障类: 保险/年金 → 风险对冲

资产配置模型:
├── 保守型: 现金30%/固收50%/权益15%/另类5%
├── 稳健型: 现金20%/固收40%/权益30%/另类10%
├── 平衡型: 现金10%/固收30%/权益45%/另类15%
├── 成长型: 现金5%/固收20%/权益55%/另类20%
└── 激进型: 现金5%/固收10%/权益65%/另类20%

配置原则:
├── 分散投资: 不要把鸡蛋放在一个篮子里
├── 定期再平衡: 每年或偏离目标5%时调整
├── 长期持有: 避免频繁交易
└── 成本控制: 关注费率/税收
```

### 投资策略

```
投资原则:
├── 长期投资: 时间是朋友/复利效应
├── 分散投资: 降低单一资产风险
├── 定投策略: 平滑成本/避免择时
├── 价值投资: 买入低估优质资产
└── 风险控制: 不懂不投/控制仓位

定投策略:
├── 适用场景: 长期投资/波动市场
├── 投资频率: 每月/每两周
├── 投资金额: 固定金额或收入比例
├── 止盈策略: 目标收益率/估值止盈
└── 心态管理: 不因短期波动停止定投

估值指标:
├── PE市盈率: 股价/每股收益 → 低PE可能被低估
├── PB市净率: 股价/每股净资产 → 低PB可能被低估
├── PEG: PE/盈利增长率 → <1可能被低估
└── 股息率: 每股股息/股价 → 高股息率更安全
```

### 财务自由路径

```
财务自由阶段:
├── 阶段1: 应急储备(3-6个月支出)
├── 阶段2: 债务清零(高息债务优先)
├── 阶段3: 保险配置(风险对冲)
├── 阶段4: 财务安全(被动收入覆盖基本支出)
└── 阶段5: 财务自由(被动收入覆盖理想生活支出)

财务自由公式:
被动收入 ≥ 生活支出 = 财务自由

被动收入来源:
├── 投资收益: 股息/利息/资本利得
├── 租金收入: 房产出租
├── 版税收入: 知识产权
├── 企业分红: 股权投资
└── 其他收入: 副业/兼职

4%法则:
年度安全提取率 = 投资组合 × 4%
所需资产 = 年度支出 ÷ 4%
例: 年支出20万 → 需500万资产
```

### 风险管理

```
保险配置:
├── 医疗险: 百万医疗/中端医疗 → 大病医疗费用
├── 重疾险: 保额3-5倍年收入 → 收入补偿
├── 意外险: 保额10倍年收入 → 意外身故/伤残
├── 定期寿险: 保额覆盖负债+5倍年收入 → 家庭责任
└── 年金险: 养老规划 → 稳定现金流

应急储备:
├── 金额: 3-6个月生活支出
├── 形式: 活期/货币基金/短期理财
├── 用途: 失业/疾病/意外
└── 补充: 用后及时补充

风险对冲:
├── 分散投资: 不同资产类别/地区/行业
├── 对冲工具: 期权/期货(专业投资者)
├── 现金储备: 保持流动性
└── 保险保障: 转移重大风险
```

### 税务规划

```
个人所得税优化:
├── 专项附加扣除: 子女教育/房贷利息/赡养老人等
├── 年终奖优化: 单独计税vs并入综合所得
├── 公积金: 提高公积金缴纳比例
├── 商业健康险: 税前扣除2400元/年
└── 个人养老金: 税前扣除12000元/年

投资税务:
├── 股票分红: 持有>1年免税
├── 基金分红: 免税
├── 股票交易: 印花税0.1%(卖出)
└── 房产交易: 契税/增值税/个税
```
