## 📚 核心框架

### 文案心理学原理

```
AIDA模型:
Attention注意 → Interest兴趣 → Desire欲望 → Action行动

Attention注意: 3秒内抓住眼球
├── 标题决定80%点击率
├── 方法: 痛点/悬念/反直觉/数据
└── 示例: "月薪3000到月薪3万我只做对了这件事"

Interest兴趣: 激发阅读欲望
├── 痛点共鸣
├── 好奇心驱动
├── 利益承诺
└── 故事开场

Desire欲望: 建立购买渴望
├── 利益点展示
├── 社会证明
├── 稀缺性
└── 恐惧诉求

Action行动: 促成明确行动
├── CTA按钮
├── 限时优惠
├── 零风险承诺
└── 简化流程

PAS框架:
Problem问题 → Agitation煽动 → Solution解决方案

Problem: 提出问题
└── "你是否每天早上醒来感到疲惫不堪?"

Agitation: 放大痛苦
└── "咖啡越喝越多但效果越来越差下午三点又开始犯困..."

Solution: 给出方案
└── "XX能量饮采用天然成分让你一整天保持清醒"
```

### Slogan创作

```
Slogan创作法则:
├── 短: 理想长度4-8个字
├── 有记忆点: 押韵/对比/双关/反转
├── 传达差异化: 不是"我们很好"而是"我们和别人不一样"
└── 情感共鸣: 触发用户情感

经典公式:
├── 利益+情感: "怕上火喝王老吉"
├── 对比式: "不是所有牛奶都叫特仑苏"
├── 行动号召: "Just Do It"
├── 身份认同: "我命由我不由天"
├── 问题解决: "充电5分钟通话2小时"
└── 场景绑定: "困了累了喝红牛"

创作步骤:
├── 提炼核心价值: 品牌最核心的差异点
├── 寻找情感连接: 与用户的情感共鸣点
├── 尝试多种表达: 至少产出10个备选
└── 测试记忆度: 用户能否记住和复述
```

### 转化型文案

```
落地页文案结构:
├── Hero区: 一句话价值主张 + 主视觉 + CTA
├── 痛点共鸣: "你是不是也..."(3个用户真实痛点)
├── 产品介绍: 核心功能 + 用户利益
├── 社会证明: 用户评价/案例/数据/媒体背书
├── FAQ: 预处理异议
└── 最终CTA: 再次强调价值 + 行动号召

产品详情页文案:
├── 标题: 核心卖点 + 利益承诺
├── 副标题: 补充说明/信任背书
├── 产品描述:
│   ├── 功能介绍 → 用户利益
│   ├── 规格参数 → 使用场景
│   └── 材质工艺 → 品质保证
├── 用户评价: 真实用户反馈
├── 购买保障: 退换货/售后
└── 紧迫感: 库存/限时优惠

广告文案:
├── 标题: 3秒抓住注意力
├── 正文: 痛点 → 方案 → 利益
├── CTA: 明确行动指令
└── A/B测试: 多版本测试优化
```

### 社交媒体文案

```
小红书文案:
├── 标题: 吸引眼球/包含关键词
├── 开头: 痛点共鸣/悬念设置
├── 正文:
│   ├── 分点罗列(1/2/3...)
│   ├── 表情符号增加可读性
│   └── 真实体验分享
├── 结尾: 互动引导/关注提示
└── 标签: #关键词 #话题

公众号文案:
├── 标题: 10-20字/包含核心信息
├── 摘要: 吸引点击/补充信息
├── 正文:
│   ├── 开头: 痛点/故事/数据
│   ├── 中间: 分段清晰/小标题
│   └── 结尾: 总结/行动号召
└── 排版: 图文结合/重点加粗

抖音文案:
├── 前3秒: 黄金开场/抓住注意力
├── 中间: 快节奏/信息密度高
├── 结尾: 引导互动/关注点赞
└── 字幕: 关键信息字幕强化

朋友圈文案:
├── 字数: 6行以内(避免折叠)
├── 内容: 真实/有温度/有价值
├── 配图: 1张/3张/6张/9张
└── 互动: 提问/投票/话题
```

### 文案优化技巧

```
标题优化:
├── 数字法: "3个方法让你..."
├── 提问法: "为什么..."
├── 对比法: "A vs B"
├── 悬念法: "你绝对想不到..."
├── 利益法: "如何获得..."
└── 紧迫法: "限时/最后机会"

文案可读性:
├── 短句为主: 每句不超过20字
├── 分段清晰: 每段3-5行
├── 小标题: 方便快速浏览
├── 重点突出: 加粗/颜色标记
└── 图文结合: 图片辅助理解

文案测试:
├── A/B测试: 多版本对比
├── 用户反馈: 收集用户意见
├── 数据分析: 点击率/转化率
└── 持续优化: 根据数据迭代
```

### 品牌故事

```
品牌故事要素:
├── 起源: 品牌如何诞生
├── 使命: 为什么存在
├── 价值观: 相信什么
├── 差异化: 有什么不同
└── 愿景: 要去向哪里

品牌故事结构:
├── 开场: 引发共鸣的场景
├── 冲突: 遇到的问题/挑战
├── 解决: 如何解决问题
├── 成果: 取得的成就
└── 愿景: 未来的目标

品牌故事技巧:
├── 真实: 真实的故事最打动人
├── 情感: 触发用户情感共鸣
├── 人物: 有具体的人物和细节
├── 冲突: 有挑战和克服过程
└── 启发: 给用户带来启发
```
