## 📚 核心设定

### 艾瑟兰大陆

**地理**: 北境冰原(冰霜巨龙) | 中央平原(艾瑟王国) | 西部森林(精灵之森) | 南方群岛(龙岛) | 东部山脉(矮人王国)

**地标**: 世界树 | 龙眠谷 | 魔法学院 | 圣光圣殿 | 深渊之门

**历史**: 创世纪元 → 第一纪元(黄金时代) → 第二纪元(黑暗战争) → 第三纪元(重建/当前)

**势力**:
| 势力 | 特点 |
|------|------|
| 艾瑟兰王国 | 人类主导/骑士团 |
| 精灵王国 | 月精灵/木精灵/暗精灵 |
| 矮人王国 | 锻造大师/熔炉之城 |
| 龙族 | 金属龙(善)/彩色龙(恶) |
| 魔法学院 | 奥术研究/法师议会 |

### 魔法体系

**来源**: 元素魔法 | 光暗魔法 | 自然魔法 | 奥术魔法

**等级**: 学徒 → 学者 → 大师 → 宗师 → 传奇

**职业**: 法师(研究) | 术士(天赋) | 牧师(神圣) | 德鲁伊(自然) | 吟游诗人(音乐) | 魔剑士(武器)

**学院**: 初级(4年基础) → 高级(4年专业) → 大师(研究禁忌)

### 种族设定

| 种族 | 特点 | 寿命 |
|------|------|------|
| 人类 | 适应性强/创造力 | 80-100年 |
| 精灵 | 魔法天赋/长寿 | 500-1000年 |
| 矮人 | 锻造天赋/坚韧 | 300-400年 |
| 龙族 | 强大智慧 | 数千年 |
| 兽人 | 强壮勇猛 | 60-80年 |

**关系**: 人类↔精灵(友好) | 精灵↔矮人(中立) | 光明↔黑暗(敌对)

## 🎯 创作模板

### 冒险故事框架

```
第一章: 命运开始/接受任务/组建队伍
第二章: 试炼成长/获得力量/发现真相
第三章: 危机挑战/遭遇强敌/内心挣扎
第四章: 最终决战/面对黑暗/牺牲抉择
第五章: 胜利重生/世界改变/新的开始
```

### 角色组合

```
主角: 骑士/法师/游侠
导师: 智者/老战士
同伴: 精灵/矮人/其他
反派: 黑暗领主/邪恶法师
```

## 💡 核心原则

- **内在逻辑**: 世界一致性/规则不随意打破
- **细节真实**: 小细节带来大真实感
- **冲突平衡**: 冲突推动故事/平衡创造张力
- **文化灵魂**: 文化决定行为/赋予种族灵魂
- **与时俱进**: 创新让世界独特/融合新元素
