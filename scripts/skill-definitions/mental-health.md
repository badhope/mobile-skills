## 📚 核心框架

### CBT认知行为疗法

```
认知三角:
思考Thoughts → 情绪Feelings → 行为Behaviors
打破循环: 改变思维(最有效)/改变行为(较容易)/情绪随之改变

十大认知扭曲:
├── 非黑即白: "一次失败=彻底失败者"
├── 过度概括: "一次被拒=永远不被接纳"
├── 心理滤镜: 只看到负面忽略正面
├── 否定正面: "做得好只是运气好"
├── 妄下结论: 读心术"他肯定讨厌我"/末日思维"肯定要出事"
├── 夸大缩小: 夸大缺点缩小成就
├── 情绪推理: "我感觉是这样所以这就是事实"
├── "应该"陈述: "我应该做到完美"
├── 贴标签: "我是个失败者"而非"我这次失败了"
└── 个人化: 全部归因于自己

ABCDE技术:
A触发事件 → B信念 → C结果 → D质疑 → E新效果
```

### 焦虑管理

```
急性焦虑发作干预(4步):
1. 承认命名: "我现在感到非常焦虑这是正常的生理反应"
2. 地面化Grounding: 5-4-3-2-1法(看5样/摸4种/听3种/闻2种/尝1种)
3. 呼吸调控: 盒式呼吸(吸气4s-憋气4s-呼气4s-憋气4s)×4轮
4. 自我对话: "这个感觉是暂时的/我曾经挺过类似的情况/我现在是安全的"

长期焦虑管理:
├── 担忧时间: 每天15分钟固定"担忧时间"/其他时间写下来等这段时间处理
├── 暴露阶梯: 把害怕的事按焦虑程度0-10排序/从最低开始逐步暴露
├── 不确定性耐受: 练习接受"不知道"/学会在不确定中行动
└── 生活方式: 运动/睡眠/咖啡因限制/酒精避免

焦虑自测GAD-7:
├── 0-4分: 无焦虑
├── 5-9分: 轻度焦虑
├── 10-14分: 中度焦虑
└── 15-21分: 重度焦虑
```

### 情绪管理

```
情绪识别:
├── 基本情绪: 喜/怒/哀/惧/惊/厌
├── 复合情绪: 羞耻/内疚/嫉妒/骄傲/希望
└── 情绪强度: 轻微/中等/强烈

情绪调节策略:
├── 认知重评: 改变对事件的解释
├── 注意转移: 转移注意力到其他事物
├── 情绪表达: 适当表达情绪/倾诉/写作
├── 身体调节: 运动/深呼吸/放松训练
└── 问题解决: 解决引发情绪的问题

愤怒管理:
├── 识别愤怒信号: 身体紧张/心跳加速/呼吸急促
├── 暂停技术: 离开现场/数到10/深呼吸
├── 表达愤怒: 用"我"语句表达感受和需求
└── 问题解决: 找到愤怒的原因并解决

抑郁管理:
├── 行为激活: 增加愉快活动/即使不想做也去做
├── 认知重构: 识别消极思维/挑战不合理信念
├── 社会支持: 与亲友保持联系/寻求帮助
├── 规律作息: 保持规律睡眠/饮食/运动
└── 专业帮助: 症状持续>2周/影响功能/有自杀想法
```

### 正念冥想

```
正念核心要素:
├── 专注当下: 关注此时此刻
├── 不评判: 接纳而不评判
├── 初学者心态: 保持好奇和开放
└── 不执着: 观察而不执着

正念练习方法:
├── 呼吸冥想: 专注呼吸/走神时温和地拉回
├── 身体扫描: 从头到脚感受身体各部位
├── 正念行走: 专注行走时的身体感受
├── 正念饮食: 慢慢吃/专注食物的味道和口感
└── 日常正念: 洗碗/洗澡/走路时保持觉察

正念练习时间:
├── 初学者: 每天5-10分钟
├── 进阶: 每天15-30分钟
└── 高级: 每天30-60分钟

正念效果:
├── 短期: 减轻压力/改善情绪/提高专注
├── 中期: 降低焦虑/改善睡眠/增强自我觉察
└── 长期: 改变大脑结构/提升幸福感/增强心理韧性
```

### 心理韧性

```
心理韧性四要素:
├── 乐观: 看到积极面/相信未来会更好
├── 认知灵活性: 多角度看问题/适应变化
├── 社会支持: 建立支持网络/寻求帮助
└── 自我效能: 相信自己有能力应对

成长型思维vs固定型思维:
├── 固定型: 能力是固定的/害怕失败/回避挑战
└── 成长型: 能力可以发展/拥抱挑战/从失败中学习

提升心理韧性:
├── 建立支持系统: 家人/朋友/同事/专业人士
├── 培养乐观思维: 寻找积极面/感恩练习
├── 发展应对技能: 问题解决/情绪调节/沟通技巧
├── 保持身体健康: 运动/睡眠/营养
└── 寻找意义: 在困难中找到意义和成长

挫折应对:
├── 接受现实: 承认挫折的存在
├── 表达情绪: 允许自己感到难过/愤怒
├── 寻求支持: 与信任的人倾诉
├── 重新评估: 从不同角度看问题
└── 采取行动: 制定计划/逐步前进
```

### 压力管理

```
压力识别:
├── 身体信号: 头痛/肌肉紧张/疲劳/失眠
├── 情绪信号: 焦虑/易怒/情绪低落
├── 认知信号: 注意力不集中/记忆力下降
└── 行为信号: 食欲改变/社交回避/拖延

压力管理策略:
├── 问题导向: 解决压力源/改变可控因素
├── 情绪导向: 调节情绪反应/接受不可控因素
├── 社会支持: 寻求他人帮助/倾诉
└── 自我照顾: 运动/睡眠/爱好/放松

时间管理减压:
├── 优先级排序: 重要紧急矩阵
├── 学会说"不": 拒绝不合理要求
├── 委托他人: 不必事事亲力亲为
└── 预留缓冲: 不要排得太满

工作生活平衡:
├── 设定边界: 工作时间/家庭时间/个人时间
├── 定期休假: 强制休息/充电
├── 培养爱好: 工作之外的兴趣
└── 正念生活: 活在当下/享受过程
```
