## 📚 核心框架

### CEO五大职责

```
CEO核心职责:
├── 战略制定: 愿景使命 → 3-5年规划 → 复盘调整
├── 组织建设: 核心团队 → 组织架构 → 人才梯队
├── 资源配置: 投资决策 → 短期长期平衡 → 融资并购
├── 外部关系: 投资人 → 董事会 → 客户伙伴
└── 文化塑造: 以身作则 → 核心价值观 → 决策原则

时间分配: 战略25% | 团队30% | 外部25% | 运营15% | 成长5%
```

### 战略决策框架

```
战略三层次:
├── 公司战略: 做什么业务/不做什么业务
├── 业务战略: 如何竞争/差异化优势
└── 职能战略: 市场/产品/技术/运营支撑

战略规划流程:
├── 环境分析: PESTEL/行业趋势/竞争格局
├── 内部分析: 资源能力/核心竞争力
├── 战略选择: 增长/稳定/收缩
└── 战略执行: 目标分解/资源配置/监控调整

决策工具:
├── SWOT分析: 优势/劣势/机会/威胁
├── 波特五力: 行业竞争分析
├── BCG矩阵: 业务组合分析
└── 情景规划: 多情景推演

重大决策流程:
├── 问题定义: 明确决策问题
├── 信息收集: 收集相关数据
├── 方案生成: 提出多个备选方案
├── 方案评估: 分析利弊/风险
├── 决策制定: 选择最优方案
└── 执行跟踪: 执行并监控结果
```

### 组织管理框架

```
组织设计:
├── 结构: 职能型/事业部/矩阵式/扁平化
├── 流程: 决策流程/协作流程/信息流程
├── 机制: 激励机制/考核机制/晋升机制
└── 文化: 价值观/行为准则/工作氛围

团队建设:
├── 招聘: 人才画像/面试评估/背景调查
├── 培养: 培训体系/导师制/轮岗
├── 激励: 薪酬/股权/成长空间
└── 留任: 文化认同/发展机会/工作环境

文化塑造:
├── 定义: 核心价值观/行为准则
├── 传播: 招聘/培训/日常沟通
├── 落地: 制度/流程/决策
└── 强化: 榜样/仪式/奖励

人才梯队:
├── 高潜识别: 绩效+潜力评估
├── 培养计划: 轮岗/项目/导师
├── 继任计划: 关键岗位继任者
└── 保留机制: 发展机会/激励
```

### 高管沟通

```
董事会汇报:
├── 汇报频率: 季度/年度
├── 汇报内容:
│   ├── 战略执行进展
│   ├── 财务业绩
│   ├── 重大风险
│   └── 下阶段计划
├── 汇报技巧:
│   ├── 数据驱动
│   ├── 简洁明了
│   ├── 提前沟通
│   └── 主动披露问题
└── 决策请求: 明确需要董事会决策的事项

投资人沟通:
├── 定期沟通: 季度报告/年度会议
├── 沟通内容:
│   ├── 业务进展
│   ├── 财务数据
│   ├── 战略规划
│   └── 风险挑战
├── 透明原则: 好消息坏消息都要说
└── 关系维护: 定期拜访/主动汇报

媒体沟通:
├── 发言原则: 统一口径/授权发言
├── 危机应对: 快速响应/真诚沟通
└── 形象管理: 个人品牌/企业品牌
```

### 领导力发展

```
领导力模型:
├── 愿景能力: 描绘未来/激发热情
├── 战略能力: 分析判断/决策规划
├── 执行能力: 组织协调/推动落地
├── 人才能力: 识人用人/培养团队
└── 变革能力: 推动变革/适应变化

领导风格:
├── 变革型领导: 激发愿景/激励变革
├── 服务型领导: 服务团队/赋能他人
├── 民主型领导: 参与决策/集思广益
└── 情境型领导: 根据情境调整风格

自我管理:
├── 时间管理: 要事第一/授权他人
├── 情绪管理: 压力管理/情绪调节
├── 持续学习: 阅读/培训/反思
└── 工作生活平衡: 健康/家庭/爱好

领导力提升路径:
├── 360度反馈: 了解自己的优势和不足
├── 导师指导: 寻找经验丰富的导师
├── 实践历练: 承担挑战性项目
├── 反思总结: 定期复盘和反思
└── 持续学习: 阅读和培训
```

### 绩效管理

```
目标管理:
├── 战略目标: 3-5年战略目标
├── 年度目标: 年度OKR/KPI
├── 季度目标: 季度OKR
└── 个人目标: 个人绩效目标

绩效评估:
├── 评估周期: 季度/年度
├── 评估维度:
│   ├── 业绩达成
│   ├── 能力提升
│   └── 行为表现
├── 评估方法:
│   ├── 自评
│   ├── 上级评估
│   └── 360度反馈
└── 结果应用: 晋升/调薪/淘汰

激励机制:
├── 短期激励: 年终奖/项目奖
├── 长期激励: 股权/期权
├── 非物质激励: 认可/成长/自主权
└── 激励原则: 公平/透明/差异化
```
