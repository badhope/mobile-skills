## 📚 核心框架

### 销售战略框架

```
销售战略三要素:
├── 目标市场: 客户画像ICP/市场细分/优先级/市场规模TAM/SAM/SOM
├── 销售模式: Inbound(入站)/Outbound(出站)/混合模式
└── 渠道策略: 直销/渠道/线上/混合渠道

销售目标制定:
├── 目标分解: 年度→季度→月度→周→个人
├── 目标类型: 营收/新客/续约/过程目标
└── SMART原则: 具体/可衡量/可达成/相关/有时限

销售预测方法:
├── 漏斗预测法: 预测 = 漏斗金额 × 转化率
├── 加权预测法: 预测 = Σ(机会金额 × 成交概率)
└── 历史趋势法: 预测 = 历史同期 × 增长率
```

### 销售团队管理

```
销售能力成长阶梯:
├── S1初级销售(0-1年): 产品知识/销售流程/基础沟通
├── S2中级销售(1-3年): 独立销售/客户维护/异议处理
├── S3高级销售(3-5年): 大客户/复杂销售/销售策略
├── S4资深销售(5年+): 战略客户/销售方法论/行业影响力
└── S5销售经理: 团队管理/销售战略/人才培养

薪酬激励体系:
├── 薪酬结构: 基本工资 + 提成 + 奖金
├── 阶梯提成: 0-80%目标5% | 80-100%8% | 100-120%12% | 120%+15%
├── 新客提成: 高于老客(鼓励开拓)
├── 奖金设计: 月度冠军/季度冠军/年度MVP/团队达成奖
└── 非金钱激励: 荣誉/晋升/培训/团建

团队管理机制:
├── 每日: 早会/晚报
├── 每周: 周会/1on1/培训分享
├── 每月: 月度会议/团建/绩效评估
└── 每季: 季度总结/晋升评估/激励调整
```

### 大客户销售

```
大客户销售七步法:
1. 客户研究(1-2周): 公司背景/决策链/痛点/竞品/预算
2. 建立关系(2-4周): 找关键人/初步联系/获取信任
3. 需求挖掘(1-2周): 深度访谈/痛点确认/决策标准
4. 方案设计(1-2周): 定制方案/ROI分析/案例准备
5. 方案呈现(1周): 正式提案/Demo演示/获取反馈
6. 商务谈判(1-2周): 价格谈判/条款协商/合同起草
7. 成交交付(1-2周): 合同签署/项目启动/关系维护

B2B决策链:
├── 决策者: 最终签字的人/老板
├── 影响者: 有话语权的人/技术/财务
├── 使用者: 实际使用产品的人
├── 把关者: 控制信息流的人/助理/采购
└── 发起者: 提出需求的人

谈判技巧:
├── 准备充分: 了解底线/设定底线/准备BATNA
├── 创造价值: 不只价格/找共同点/扩大蛋糕
├── 让步策略: 小幅让步/有条件让步/逐步让步
├── 应对价格: 强调价值/分解成本/对比竞品
└── 达成协议: 总结共识/明确下一步/书面确认
```

### 销售漏斗管理

```
销售漏斗阶段:
线索Leads(100%) → 合格线索MQL(30%) → 销售线索SQL(15%) → 机会Opportunity(6%) → 成交Deal(3%)

关键指标:
├── 线索数量: 进入漏斗的线索数
├── 转化率: 各阶段转化率
├── 流失率: 各阶段流失原因
├── 平均成交周期: 从线索到成交的时间
├── 平均客单价: 成交金额
└── 漏斗金额: 各阶段金额总和

漏斗优化:
├── 线索不足: 增加营销投入/拓展渠道
├── 质量差: 优化ICP/改进筛选标准
├── 转化低: 改进销售话术/加强培训
├── 成交率低: 优化方案/加强Demo
└── 流失多: 改进谈判技巧/灵活定价
```

## 💼 实战模板

### 销售团队搭建模板

```
一、团队规划
├── 组织架构: 销售总监/销售经理/销售代表/销售支持
├── 招聘计划: 分批次招聘时间表
└── 招聘标准: 经验/能力/薪资

二、培训体系
├── 新人培训(2周): 产品市场/销售技能/考核
├── 持续培训: 每周分享/每月更新/每季方法论
└── 外部培训: 每年2次

三、激励体系
├── 薪酬结构: 基本工资/提成/奖金
├── 晋升通道: 销售代表→高级销售→销售经理→销售总监
└── 非金钱激励: 荣誉/培训/团建
```

### 大客户销售模板

```
一、客户研究
├── 客户背景: 公司/规模/预算/现状
├── 决策链分析: 决策者/影响者/使用者/把关者/发起者
├── 痛点识别: 主要痛点/优先级
└── 竞品分析: 竞品优劣势/我方优势

二、销售策略
├── 切入点: 从哪个角色开始
├── 价值主张: 对各角色的价值
├── 差异化策略: 如何区别竞品
└── POC计划: 概念验证方案

三、执行计划
├── 时间线: 各阶段时间安排
├── 关键里程碑: 重要节点
├── 资源需求: 需要的支持
└── 风险预案: 可能的问题和应对
```
