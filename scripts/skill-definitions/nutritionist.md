## 📚 核心框架

### 营养素分类

```
宏量营养素:
├── 碳水化合物: 主要能量来源/谷物薯类水果/占总能量50-65%
├── 蛋白质: 组织修复酶免疫/肉蛋奶豆类/0.8-1.2g/kg体重
└── 脂肪: 能量储存激素合成/油脂坚果肉类/占总能量20-30%

微量营养素:
├── 维生素: 脂溶性(A/D/E/K)/水溶性(B族/C)/代谢调节
└── 矿物质: 常量(钙磷镁钠钾)/微量(铁锌碘硒)/结构代谢调节

其他:
├── 水: 代谢介质/体温调节
├── 膳食纤维: 肠道健康/每日25-30g
└── 植物化学物: 抗氧化/保健功能
```

### 能量代谢

```
能量消耗构成:
├── 基础代谢(BMR): 占60-75%/维持基本生命活动
├── 体力活动: 占15-30%/日常活动运动
└── 食物热效应: 占10%/消化吸收代谢

BMR计算:
男性: BMR = 10×体重 + 6.25×身高 - 5×年龄 + 5
女性: BMR = 10×体重 + 6.25×身高 - 5×年龄 - 161

活动系数:
├── 久坐: 1.2
├── 轻度活动: 1.375
├── 中度活动: 1.55
├── 重度活动: 1.725
└── 极重度活动: 1.9

总能量消耗 = BMR × 活动系数
```

### 膳食评估

```
膳食调查方法:
├── 24小时回顾法: 回忆过去24小时所有食物
├── 膳食记录法: 记录3-7天所有食物
├── 食物频率问卷: 评估长期膳食模式
└── 膳食史法: 了解饮食习惯和偏好

营养评估指标:
├── 人体测量: BMI/腰围/体脂率
├── 生化指标: 血常规/血脂/血糖/维生素
├── 临床检查: 营养缺乏症状
└── 膳食分析: 能量/营养素摄入量

BMI标准:
├── <18.5: 体重过轻
├── 18.5-23.9: 正常
├── 24-27.9: 超重
└── ≥28: 肥胖
```

### 膳食指南

```
中国居民膳食指南(2022):
├── 食物多样: 每日12种以上/每周25种以上
├── 合理搭配: 粗细搭配/荤素搭配
├── 适量吃鱼禽蛋瘦肉: 每日120-200g
├── 少盐少油控糖: 盐<5g/油25-30g/糖<25g
├── 规律进餐: 三餐定时定量
├── 足量饮水: 每日1500-1700ml
├── 会烹会选: 学会阅读食品标签
└── 公筷分餐: 预防疾病传播

膳食宝塔:
├── 第一层: 谷薯类250-400g
├── 第二层: 蔬菜300-500g/水果200-350g
├── 第三层: 畜禽肉40-75g/水产品40-75g/蛋类40-50g
├── 第四层: 奶类300-500g/大豆坚果25-35g
└── 第五层: 油25-30g/盐<5g
```

### 特殊人群营养

```
孕妇营养:
├── 孕早期: 叶酸400μg/d/预防神经管畸形
├── 孕中期: 铁24mg/d/钙1000mg/d
├── 孕晚期: 铁29mg/d/蛋白质增加15g
└── 哺乳期: 能量+500kcal/蛋白质+20g

儿童青少年:
├── 能量需求高/生长发育需要
├── 蛋白质需求高/组织生长
├── 钙需求高/骨骼发育
└── 铁需求高/血容量增加

老年人:
├── 能量需求减少/代谢降低
├── 蛋白质需求增加/肌肉流失
├── 钙需求增加/骨质疏松
├── 维生素D需求增加/吸收减少
└── 水分需求增加/渴感降低
```

### 慢病营养干预

```
糖尿病营养:
├── 控制总能量: 维持理想体重
├── 碳水化合物: 低GI食物/占总能量50-60%
├── 蛋白质: 1.0-1.2g/kg/d
├── 脂肪: <30%/限制饱和脂肪
└── 膳食纤维: 25-30g/d

高血压营养:
├── 限盐: <5g/d
├── 增加钾: 蔬菜水果
├── 增加钙: 奶类豆类
├── DASH饮食: 蔬菜水果全谷物
└── 限制饮酒

高血脂营养:
├── 限制总脂肪: <30%
├── 限制饱和脂肪: <7%
├── 限制胆固醇: <300mg/d
├── 增加膳食纤维: 25-30g/d
└── 增加不饱和脂肪: 鱼类坚果
```

### 体重管理

```
减重原则:
├── 能量负平衡: 摄入<消耗
├── 循序渐进: 每周减0.5-1kg
├── 营养均衡: 不节食/不偏食
└── 行为改变: 饮食习惯/运动习惯

减重膳食:
├── 能量减少: 每日减少500-750kcal
├── 蛋白质: 1.2-1.5g/kg/d/维持肌肉
├── 碳水化合物: 低GI/高纤维
├── 脂肪: 适量/不饱和脂肪为主
└── 蔬菜: 充足/增加饱腹感

增重膳食:
├── 能量增加: 每日增加300-500kcal
├── 少量多餐: 每日5-6餐
├── 营养密度高: 选择高营养食物
└── 适量运动: 增加肌肉而非脂肪
```
