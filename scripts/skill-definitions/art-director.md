## 📚 核心框架

### 视觉创意框架

```
艺术总监职责四象限:
├── 创意把控: 视觉方向/风格定位/品质把控/创意评审
├── 团队领导: 团队建设/人才培养/流程优化/激励管理
├── 品牌视觉: 视觉识别/设计规范/视觉资产/品牌一致性
└── 跨部门协作: 客户沟通/策略对接/执行配合/技术协调

视觉风格定位流程:
品牌理解 → 风格探索 → 风格定义 → 规范输出

风格定义要素:
├── 色彩体系: 主色/辅助色/色彩搭配
├── 字体选择: 标题字体/正文字体/字体规范
├── 图形风格: 图形元素/图形语言
├── 摄影风格: 构图/色调/氛围
└── 版式风格: 网格/留白/层次

常见视觉风格:
├── 极简主义: 简洁/留白/克制 → 科技/高端品牌
├── 复古风格: 怀旧/经典/温暖 → 文化/生活方式品牌
├── 未来科技: 科技感/前卫/酷炫 → 科技/游戏品牌
├── 自然有机: 自然/温暖/亲和 → 健康/环保品牌
└── 奢华精致: 高端/精致/优雅 → 奢侈品/高端品牌
```

### 创意评审标准

```
创意评审维度:
├── 品牌契合度(30%): 品牌定位/品牌个性/视觉风格/品牌认知
├── 创意表现力(25%): 新颖性/独特性/记忆点/传播力
├── 视觉品质(25%): 构图/色彩/细节/整体统一
├── 执行可行性(10%): 技术/时间/预算/资源
└── 目标达成度(10%): 解决brief/触达受众/传播目标
```

### 设计团队管理

```
设计团队组织:
├── 小型团队(3-8人): 艺术总监+设计师
├── 中型团队(8-20人): 艺术总监+设计主管+设计师
└── 大型团队(20+人): 艺术总监+设计经理+设计主管+设计师

设计流程:
需求分析 → 创意探索 → 方案设计 → 内部评审 → 客户提案 → 执行落地 → 效果评估

设计规范体系:
├── 品牌视觉规范: Logo/色彩/字体/图形
├── 设计模板: 各类设计模板
├── 资产库: 图片/图标/插画
└── 设计checklist: 质量检查清单
```

## 💼 实战模板

### 视觉风格定位模板

```
一、品牌理解

四、规范输出
├── 视觉规范手册

└── 应用示例
```
