## 📚 核心框架

### 运营战略框架

```
运营体系架构:
├── 战略层: 目标制定/策略规划/资源分配
├── 执行层: 用户运营/内容运营/活动运营/产品运营
└── 支撑层: 数据分析/工具平台/流程制度

运营目标金字塔:
├── 北极星指标: 核心目标(如MAU)
├── 一级指标: 新增用户/留存用户/召回用户
└── 二级指标: 渠道成本/留存率/召回转化率

OKR目标管理:
Objective: 提升用户留存
├── KR1: 次日留存率从40%提升到50%
├── KR2: 7日留存率从20%提升到30%
└── KR3: 月留存率从10%提升到15%
```

### 用户运营

```
用户生命周期:
引入期(拉新) → 成长期(促活) → 成熟期(留存) → 衰退期(召回) → 流失期(分析)

各阶段运营:
├── 引入期: 目标-首次核心行为 | 策略-新手引导/首单优惠 | 指标-激活率
├── 成长期: 目标-提升频次深度 | 策略-个性化推荐/任务激励 | 指标-DAU/时长
├── 成熟期: 目标-提升付费推荐 | 策略-会员体系/推荐奖励 | 指标-付费率/推荐率
├── 衰退期: 目标-唤醒激活 | 策略-推送触达/优惠召回 | 指标-召回率
└── 流失期: 目标-分析原因 | 策略-流失调研/产品优化 | 指标-流失率

用户分层运营:
├── 价值分层: 高价值VIP服务 | 中价值成长激励 | 低价值培育引导
└── 行为分层: 活跃深度运营 | 沉默唤醒触达 | 流失分析优化

增长策略矩阵:
├── 获客增长: 渠道优化/裂变增长/内容获客/品牌获客
├── 留存增长: 产品留存/内容留存/社交留存/激励留存
├── 变现增长: 转化优化/客单提升/复购提升/新增变现
└── 推荐增长: 口碑传播/激励推荐/社交分享/KOL传播
```

### 活动运营

```
活动策划八步法:
1. 明确目标: 业务目标/用户目标/品牌目标
2. 用户分析: 用户画像/需求分析/触媒习惯
3. 创意策划: 主题/形式/规则/奖励
4. 方案设计: 流程/页面/交互/视觉
5. 资源准备: 技术/奖品/渠道/人员
6. 推广执行: 渠道投放/内容发布/用户触达/实时监控
7. 数据监控: 核心指标/异常预警/实时调整/数据记录
8. 复盘总结: 效果评估/经验总结/问题分析/优化建议

活动类型:
├── 拉新活动: 邀请有礼/新人礼包 → 新增用户数/CAC
├── 促活活动: 签到/任务/游戏 → DAU/参与率
├── 转化活动: 限时折扣/满减/秒杀 → GMV/转化率
├── 留存活动: 会员日/积分兑换 → 留存率/复购率
└── 传播活动: 话题/挑战赛/UGC → 传播量/参与量
```

### 数据运营

```
数据分析层级:
├── 描述性分析: 发生了什么? → 数据报表/可视化
├── 诊断性分析: 为什么发生? → 数据下钻/归因分析
├── 预测性分析: 未来会怎样? → 趋势预测/模型预测
└── 规范性分析: 应该怎么做? → 优化建议/决策支持

运营指标体系:
├── 用户指标: DAU/MAU/留存率/增长率/流失率
├── 业务指标: GMV/订单量/客单价/转化率/ARPU
├── 活动指标: 曝光量/参与率/转化率/ROI
└── 内容指标: 生产量/曝光量/互动量/转化率

数据驱动决策流程:
问题定义 → 数据收集 → 数据分析 → 洞察提炼 → 决策执行 → 效果验证
```

## 💼 实战模板

### 用户增长策略模板

```
一、现状分析
├── 当前数据: 新增用户/主要渠道/CAC/留存率
├── 问题诊断: 渠道/裂变/内容/留存问题
└── 增长机会: 可拓展的方向

二、增长策略
├── 渠道拓展: 各渠道预期增量
├── 裂变增长: 邀请有礼/拼团/分享解锁
├── 内容获客: SEO/内容营销/社交媒体
└── 总目标: 现有+新增

三、执行计划
├── 时间表: 各月目标
├── 预算: 各项费用明细
└── 预期CAC: 成本效益分析
```

### 活动策划模板

```
一、活动目标
├── GMV目标
├── 订单量目标
├── 新用户目标
└── 参与人数目标

二、活动方案
├── 活动主题
├── 活动时间: 预热期/爆发期/返场期
├── 活动玩法: 秒杀/满减/拼团/会员专享/抽奖
└── 资源投入: 技术/人力/广告/补贴

三、执行计划
├── 预热期: 预告/预售/加购/会员预热
├── 爆发期: 各日主题/直播/秒杀
├── 返场期: 延续优惠/补货/回馈
└── 效果评估: GMV/订单/新用户/ROI
```

## 🚀 核心原则

- 数据驱动: 数据不会骗人/能发现问题/指导优化
- 用户价值: 关注用户质量不只是数量
- 系统工程: 用户/内容/活动运营协同/形成闭环
- 懂产品: 运营和产品是共生关系
- 快速迭代: 小步快跑/快速迭代
