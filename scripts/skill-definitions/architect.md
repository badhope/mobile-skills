## 📚 核心框架

### 建筑设计流程

```
设计全流程:
前期策划 → 概念设计 → 方案设计 → 初步设计 → 施工图设计

前期策划:
├── 项目分析: 功能需求/投资预算/时间要求
├── 场地调研: 地形/气候/交通/周边环境
└── 任务书解读: 明确设计目标和约束

概念设计:
├── 设计理念: 核心思想/设计主题
├── 功能泡泡图: 功能关系/空间组织
└── 体量推敲: 建筑形态/空间体量

方案设计:
├── 平面设计: 各层平面/功能布局
├── 立面设计: 建筑外观/材料表达
├── 剖面设计: 空间关系/高度控制
└── 效果图: 视觉表达/氛围呈现
```

### 建筑三要素

```
功能Function:
├── 使用需求: 功能分区/空间大小
├── 流线组织: 人流/物流/车流
├── 空间品质: 采光/通风/视野
└── 灵活性: 适应性/可变性

技术Technology:
├── 结构系统: 框架/剪力墙/钢结构
├── 建筑设备: 暖通/给排水/电气
├── 建筑材料: 材料选择/构造做法
└── 施工技术: 施工方法/工艺要求

形式Form:
├── 空间形态: 体量/比例/尺度
├── 立面表达: 材质/色彩/肌理
├── 建筑风格: 现代主义/地域性/个性化
└── 场所精神: 文化/历史/环境
```

### 空间规划

```
功能分区原则:
├── 动静分区: 安静区与活动区分离
├── 洁污分区: 清洁区与污染区分离
├── 内外分区: 公共区与私密区分离
└── 功能相近: 相关功能靠近布置

流线设计:
├── 人流: 主要入口/水平交通/垂直交通
├── 物流: 货物运输/垃圾清运
├── 车流: 机动车/非机动车/停车
└── 流线原则: 简洁/明确/不交叉

空间组织:
├── 集中式: 核心空间+附属空间
├── 线性式: 空间沿轴线排列
├── 辐射式: 中心空间+辐射空间
├── 网格式: 规则网格组织
└── 组合式: 多种方式组合
```

### 建筑美学

```
形式美法则:
├── 统一与变化: 整体统一/局部变化
├── 比例与尺度: 黄金比例/人体尺度
├── 对称与均衡: 绝对对称/动态均衡
├── 节奏与韵律: 重复/渐变/对比
└── 对比与调和: 形式/材质/色彩对比

立面设计:
├── 构图: 立面划分/虚实关系
├── 材质: 材料选择/肌理表达
├── 色彩: 主色调/辅助色/点缀色
├── 细部: 线脚/窗台/阳台/雨棚
└── 光影: 阴影效果/光影变化

建筑风格:
├── 现代主义: 简洁/功能/理性
├── 地域性: 本土材料/传统元素
├── 高技派: 技术表达/结构外露
├── 解构主义: 破碎/非线性/动态
└── 参数化设计: 数字化/生成式
```

### 可持续设计

```
绿色建筑原则:
├── 节能: 减少能源消耗/提高能效
├── 节地: 合理利用土地/集约开发
├── 节水: 雨水收集/中水回用
├── 节材: 可持续材料/减少浪费
└── 环保: 减少污染/保护生态

被动式设计:
├── 自然采光: 窗墙比/采光井/天窗
├── 自然通风: 穿堂风/热压通风/风塔
├── 遮阳设计: 水平遮阳/垂直遮阳/活动遮阳
├── 围护结构: 保温/隔热/气密性
└── 热缓冲空间: 中庭/阳光房/双层表皮

主动式技术:
├── 太阳能: 光伏发电/太阳能热水
├── 地源热泵: 地热利用
├── 绿色屋顶: 屋顶绿化/雨水管理
└── 智能控制: 能源管理/环境监测

绿色建筑认证:
├── 中国绿色建筑评价标准: 一星/二星/三星
├── LEED: 认证级/银级/金级/铂金级
└── WELL: 健康/舒适/心理健康
```

### 设计表达

```
图纸类型:
├── 总平面图: 建筑与场地关系
├── 平面图: 各层平面布局
├── 立面图: 建筑外观
├── 剖面图: 空间关系/高度
├── 详图: 节点构造/细部做法
└── 效果图: 视觉表达/氛围

设计说明:
├── 工程概况: 项目基本信息
├── 设计依据: 规范/标准/法规
├── 设计理念: 核心思想/设计策略
├── 技术措施: 结构/设备/材料
└── 经济指标: 面积/造价/能耗

模型表达:
├── 体量模型: 空间形态推敲
├── 工作模型: 方案深化研究
├── 展示模型: 最终成果展示
└── 数字模型: BIM/参数化设计
```
