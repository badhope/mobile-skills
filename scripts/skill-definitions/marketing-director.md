## 📚 核心框架

### 市场战略框架

```
市场分析工具:
├── PEST分析: 政策/经济/社会/技术
├── 波特五力: 竞争者/进入者/替代品/供应商/客户
├── SWOT分析: 优势/劣势/机会/威胁
└── STP战略: 市场细分/目标市场/市场定位

竞争策略:
├── 市场领导者: 扩大市场/防御竞争
├── 市场挑战者: 攻击领导者/差异化竞争
├── 市场追随者: 模仿创新/细分市场
└── 市场利基者: 专注细分市场

定位策略:
├── 产品定位: 突出产品特点
├── 价格定位: 价格优势
├── 服务定位: 服务差异化
├── 情感定位: 情感共鸣
└── 场景定位: 使用场景
```

### 品牌建设框架

```
品牌定位三角:
├── 目标受众: 谁是你的客户?
├── 竞争差异: 你有什么不同?
├── 品牌承诺: 你能给客户什么?
└── 证明支撑: 为什么相信你?

品牌传播渠道:
├── 自有媒体: 官网/社交媒体/邮件/门店
├── 赢得媒体: 媒体报道/用户口碑/社交传播
└── 付费媒体: 广告投放/KOL合作/活动赞助

品牌资产模型:
├── 品牌知名度: 品牌识别/品牌回忆
├── 品牌联想: 产品属性/品牌个性
├── 品牌忠诚度: 行为忠诚/态度忠诚
├── 感知质量: 产品质量/服务质量
└── 其他资产: 专利/商标/渠道关系
```

### 营销策划框架

```
活动策划七步法:
1. 明确目标: 品牌/销售/用户目标
2. 目标人群: 用户画像/触媒习惯
3. 创意策划: 主题/创意/视觉/内容
4. 渠道选择: 线上/线下/整合传播
5. 预算规划: 媒体/制作/执行/机动
6. 执行计划: 时间/分工/资源/风险
7. 效果评估: 曝光/互动/转化/ROI

数字营销渠道:
├── 搜索引擎: 主动搜索/精准
├── 信息流: 兴趣推荐/原生
├── 社交媒体: 互动传播/口碑
├── 短视频: 内容驱动/传播快
├── 直播: 实时互动/转化高
└── KOL: 信任背书/影响大
```

### 市场增长框架

```
AARRR增长模型:
├── Acquisition 获取: 渠道/指标/优化
├── Activation 激活: 首次体验/核心功能
├── Retention 留存: 产品价值/用户习惯
├── Revenue 变现: 商业模式/付费转化
└── Referral 推荐: 口碑传播/推荐机制

获客渠道:
├── 付费获客: SEM/信息流/社交广告/KOL
├── 自然获客: SEO/内容营销/社交媒体
└── 推荐获客: 老带新/社交裂变/分销

关键指标:
├── CAC(获客成本) = 营销费用 ÷ 新增用户数
├── LTV(用户生命周期价值)
└── LTV/CAC比率: <1不可持续 | 1-3需优化 | 3-5健康 | >5可加大投入
```

## 💼 实战模板

### 品牌定位模板

```
一、市场分析
├── 行业分析: 规模/增长/趋势
├── 竞品分析: 国际品牌/国内品牌/新锐品牌
└── 用户分析: 核心用户/需求/痛点/触媒

二、品牌定位
├── 目标受众: 人口统计/心理特征/行为特征
├── 品牌定位: 一句话定位
├── 品牌承诺: 给客户的核心价值
├── 竞争差异: vs主要竞品的差异点
├── 品牌个性: 3-5个关键词
└── 品牌口号: 简洁有力的口号

三、品牌传播
├── 内容策略: 专业/用户/品牌内容
├── 渠道策略: 核心渠道/辅助渠道
├── KOL策略: 头部/腰部/KOC
└── 预算分配: 内容/KOL/广告/活动
```

### 营销活动策划模板

```
一、活动目标
├── 品牌目标: 知名度/形象改善
├── 销售目标: 销售额/获客数
└── 用户目标: 拉新/促活/留存

二、目标人群
├── 核心用户: 人口统计/心理特征
├── 次要用户: 扩展人群
└── 触媒习惯: 主要渠道

三、活动创意
├── 活动主题: 吸引人的主题
├── 核心创意: 创意概念
├── 视觉设计: 视觉风格
└── 传播内容: 关键信息

四、执行方案
├── 时间线: 预热期/爆发期/持续期
├── 渠道投放: 各渠道内容规划
├── 预算分配: 各项费用明细
└── 效果评估: 曝光/互动/转化指标
```

## 🚀 核心原则

- 用户导向: 用户是一切市场工作的起点
- 品牌长期: 品牌是长期投资不是短期消耗
- 数据驱动: 数据+创意+策略=成功
- 整合传播: 多渠道整合触达用户
- 业务导向: 市场是业务增长的引擎
