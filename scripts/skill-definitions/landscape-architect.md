**方案**: 城市公园景观设计
1. 前期分析:
   ```
   项目概况:
   ├── 用地: 15公顷
   ├── 位置: 城市中心区
   ├── 周边: 住宅/商业/学校
   ├── 现状: 废弃工厂用地
   └── 定位: 城市综合公园
   
   场地分析:
   ├── 地形: 较平坦/局部高差
   ├── 水系: 有小河穿过
   ├── 植被: 杂草/少量乔木
   ├── 建筑: 废弃厂房
   └── 交通: 三面临街
   
   需求分析:
   ├── 周边居民: 日常休闲/儿童活动
   ├── 上班族: 午休/运动
   ├── 学生: 学习/活动
   └── 游客: 观光/体验
   ```

2. 总体规划:
   ```
   设计理念:
   "城市绿心·记忆重生"

   ├── 保留工业记忆/改造利用
   ├── 创造城市生态绿心
   ├── 提供多元活动空间

   └── 水上活动
   
   服务管理区(1公顷):
   ├── 管理中心
   ├── 停车场
   └── 服务设施
   
   景观结构:
   ├── 一环: 环形主路
   ├── 两轴: 东西/南北景观轴
   ├── 三心: 文化/运动/生态中心

   ```

## 🚀 进阶技巧

- 景观设计师最重要的工作是"创造场所": 很多设计师只关注植物/但:/景观是创造场所/植物是工具/记住: 先思考空间和场所/再选择植物
- "生态"是景观的核心价值: 很多设计师忽视生态/但:/景观有生态责任/生态是未来方向/记住: 将生态融入设计/创造生态价值
- 景观设计要"懂植物": 很多设计师不懂植物/但:/植物是活的/植物需要养护/记住: 深入了解植物特性/考虑长期生长
- "时间"是景观的维度: 很多设计师忽视时间/但:/景观随时间变化/植物会长大/记住: 考虑时间维度/设计成长性
- 景观设计师要建立"个人风格": 很多设计师没有风格/但:/个人风格是竞争力/个人风格是辨识度/记住: 发展个人风格/建立个人品牌
