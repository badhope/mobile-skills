## 📚 核心框架

### STP营销战略

```
STP三步法:
Segmentation细分 → Targeting选择 → Positioning定位

市场细分:
├── 人口统计: 年龄/性别/收入/教育/职业
├── 地理位置: 国家/城市层级/城乡
├── 心理特征: 价值观/生活方式/个性/兴趣
├── 行为特征: 使用场合/使用率/忠诚度/购买决策因素
└── 输出: 3-5个有意义的细分市场

目标市场选择:
├── 评估维度: 市场规模/增长潜力/竞争强度/可达性
├── 选择策略:
│   ├── 集中单一细分: 资源有限时
│   ├── 差异化多细分: 不同产品针对不同群体
│   └── 无差异大众市场: 标准化产品
└── 用户画像Persona: 名字/面孔/生活故事

品牌定位:
├── 定位声明: 对[目标用户]而言,[品牌名]是[品类]中的[差异化点]因为[证据]
├── 差异化方向:
│   ├── 产品特性: 更快更便宜更好看
│   ├── 服务体验: 更贴心更快速
│   └── 情感联结: 更有态度更有归属感
└── 定位测试: 可信? 独特? 可持续? 用户在乎?
```

### AARRR增长漏斗

```
AARRR海盗指标:
Acquisition获取 → Activation激活 → Retention留存 → Revenue变现 → Referral推荐

Acquisition获取: 用户从哪里来?
├── 渠道: SEO/SEM/社交媒体/内容营销/口碑推荐/合作推广
├── 关键指标: CAC获客成本/渠道转化率
└── 策略: 多渠道测试/关注渠道-产品匹配度

Activation激活: 首次体验核心价值了吗?
├── Aha Moment: 用户第一次感受到"真好用"的瞬间
├── 优化首次体验: Onboarding流程/消除摩擦
└── 关键指标: 激活率/Time to Value

Retention留存: 用户会回来吗?
├── 留存曲线: D1/D7/D30留存率
├── 健康标准: SaaS产品D7>40% D30>20%
├── 留存驱动: 产品价值/习惯养成/社交锁定/转换成本
└── 召回策略: Push通知/邮件召回/流失预警

Revenue变现: 如何赚钱?
├── 商业模式: 订阅制/交易佣金/广告/免费增值/按量计费
├── 定价策略: 成本加成/价值定价/竞争导向/动态定价
└── 关键指标: ARPU/付费转化率/LTV

Referral推荐: 用户会推荐吗?
├── 推荐机制: 邀请奖励/社交分享/口碑传播
├── 关键指标: K因子/推荐转化率
└── K因子>1: 每个用户带来>1个新用户
```

### 4P营销组合

```
4P营销组合:
Product产品 → Price价格 → Place渠道 → Promotion促销

Product产品:
├── 产品层次: 核心产品/有形产品/期望产品/延伸产品/潜在产品
├── 产品组合: 宽度/长度/深度/一致性
└── 产品生命周期: 导入/成长/成熟/衰退

Price价格:
├── 定价方法: 成本导向/需求导向/竞争导向
├── 定价策略: 撇脂定价/渗透定价/心理定价
└── 价格调整: 折扣/促销/差异化定价

Place渠道:
├── 渠道类型: 直接渠道/间接渠道/多渠道
├── 渠道设计: 长度/宽度/密度
└── 渠道管理: 选择/激励/评估/冲突解决

Promotion促销:
├── 促销工具: 广告/公关/销售促进/人员推销/直复营销
├── 整合营销传播IMC: 统一信息/多触点协同
└── 促销预算: 销售百分比/竞争对等/目标任务法
```

### 品牌建设

```
品牌资产模型:
├── 品牌知名度: 认知度/回忆率
├── 品牌联想: 属性/利益/态度
├── 品牌质量感知: 可靠性/耐用性/服务
└── 品牌忠诚: 行为忠诚/态度忠诚

品牌架构:
├── 单一品牌: 所有产品用一个品牌
├── 多品牌: 不同产品用不同品牌
├── 品牌延伸: 品牌扩展到新产品
└── 联合品牌: 两个品牌合作

品牌命名:
├── 描述性: 描述产品特性(如: 舒肤佳)
├── 联想性: 引发相关联想(如: 苹果)
├── 独创性: 无实际含义(如: 柯达)
└── 命名原则: 易读/易记/易传播/法律保护

品牌视觉识别VI:
├── 品牌标志Logo
├── 品牌色彩
├── 品牌字体
├── 品牌图形
└── 应用规范
```

### 营销效果评估

```
营销ROI计算:
ROI = (收入 - 成本) / 成本 × 100%

关键指标:
├── CPM千次曝光成本: 广告每千次展示成本
├── CPC单次点击成本: 广告每次点击成本
├── CPA单次行动成本: 用户完成特定行动成本
├── CAC获客成本: 获取一个付费用户的成本
├── LTV用户终身价值: 用户生命周期总价值
└── ROAS广告支出回报率: 广告收入/广告成本

A/B测试:
├── 定义假设: 要测试什么?
├── 设计实验: 对照组vs实验组
├── 确定样本: 统计显著性
├── 执行测试: 控制变量
└── 分析结果: 数据驱动决策

归因模型:
├── 末次点击归因: 最后一个触点获得全部功劳
├── 首次点击归因: 第一个触点获得全部功劳
├── 线性归因: 所有触点平均分配
├── 时间衰减归因: 越接近转化的触点权重越高
└── 数据驱动归因: 算法自动分配权重
```
