## 📚 核心框架

### 心理咨询原则

```
咨询基本原则:
├── 保密原则: 咨询内容保密/例外情况说明
├── 自愿原则: 来访者自愿/不强制咨询
├── 中立原则: 价值中立/不评判对错
├── 助人自助: 帮助来访者自助/培养能力
└── 时间限定: 每次50-60分钟/固定频率

保密例外情况:
├── 有伤害自己的风险
├── 有伤害他人的风险
├── 虐待儿童/老人
└── 法律要求
```

### 咨询流程

```
咨询三阶段:
├── 初始访谈(1-2次): 建立关系/了解问题/评估/制定计划
├── 咨询中期: 深入探索/识别核心问题/处理情绪/促进改变
└── 咨询后期: 巩固改变/预防复发/结束咨询

首次访谈要点:
├── 建立信任关系
├── 了解主诉问题
├── 收集背景信息
├── 评估问题严重程度
├── 明确咨询目标
└── 签订咨询协议
```

### 心理咨询流派

```
主要流派:
├── 精神分析: 潜意识/童年经历/防御机制/移情
├── 认知行为(CBT): 认知重构/行为改变/暴露疗法
├── 人本主义: 无条件积极关注/共情/真诚
├── 家庭治疗: 家庭系统/家庭互动/家庭结构
└── 叙事治疗: 外化问题/重写故事/寻找例外

CBT核心模型:
情境 → 认知 → 情绪 → 行为
      ↓
   认知重构 → 改变情绪和行为
```

### 常见心理问题

```
情绪问题:
├── 焦虑: 过度担心/紧张/恐惧/回避
├── 抑郁: 情绪低落/兴趣减退/无望感
├── 愤怒: 易怒/冲动/攻击行为
└── 悲伤: 失去/哀伤过程/适应困难

人际关系问题:
├── 亲子关系: 沟通障碍/冲突/依赖
├── 伴侣关系: 信任/沟通/亲密关系
├── 职场关系: 冲突/压力/适应
└── 社交恐惧: 害怕社交/回避/自我意识过强

适应问题:
├── 生活变故: 失业/离婚/搬家
├── 角色转换: 结婚/生子/退休
├── 文化适应: 移民/跨文化
└── 创伤后应激: PTSD/闪回/回避
```

### 心理评估工具

```
常用量表:
├── 情绪评估: SAS(焦虑)/SDS(抑郁)/BDI/BAI
├── 人格评估: MMPI/16PF/EPQ/大五人格
├── 心理健康: SCL-90/心理健康问卷
├── 压力评估: 生活事件量表/压力知觉量表
└── 人际关系: 人际关系量表/社交回避量表

评估流程:
选择工具 → 施测 → 计分 → 解释 → 反馈
```

### 咨询技术

```
基本技术:
├── 倾听: 积极倾听/专注/理解
├── 共情: 理解感受/表达理解
├── 提问: 开放式提问/封闭式提问
├── 反映: 情感反映/内容反映
└── 澄清: 明确含义/确认理解

高级技术:
├── 面质: 指出矛盾/促进觉察
├── 解释: 提供新视角/深化理解
├── 自我暴露: 适度分享/建立关系
├── 即时化: 关注当下/此时此地
└── 指导: 提供具体建议/教授技能
```

### 危机干预

```
自杀风险评估:
├── 自杀意念: 是否想死/频率/强度
├── 自杀计划: 具体方法/时间/地点
├── 自杀准备: 准备工具/写遗书/安排后事
├── 自杀历史: 过去尝试/家族史
└── 保护因素: 家庭支持/治疗关系/未来希望

危机干预步骤:
评估危险 → 确保安全 → 情感支持 → 探索选择 → 制定计划 → 跟进
```
