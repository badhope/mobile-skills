## 📚 核心框架

### 数据建模

**维度建模**: 星型模式(事实表+维度表) / 雪花模式(维度再分解) / SCD缓慢变化维

**分层架构**: ODS(贴源) → DWD(明细清洗) → DWS(汇总聚合) → ADS(应用宽表)

**ETL实践**: CDC增量捕获 / 幂等性处理 / 血缘追踪 / DQ检查

### 可视化设计

**图表选择**:
| 目的 | 图表 |
|------|------|
| 比较 | 柱状图/条形图 |
| 趋势 | 折线图/面积图 |
| 分布 | 箱线图/直方图 |
| 构成 | 饼图/堆叠柱/瀑布图 |
| 关系 | 散点图/气泡图/热力图 |

**视觉编码**: 位置 > 长度 > 角度 > 面积 > 色相 > 纹理

**色彩规范**: 感知均匀色阶(Viridis/Plasma) / 色盲友好

### KPI指标体系

**北极星指标NSM**: SaaS=NRR / 电商=GMV / 社交=DAU / 内容=时长

**漏斗分析**: 转化率/跳出率/退出率/停留时长

**同期群分析**: 留存曲线/LTV/CAC/回本周期

## 💡 关键原则

- **数据-墨水比**: 删除无意义装饰(3D效果/过度渐变)
- **故事线叙事**: 现状→问题→原因→建议→预期效果
- **统一指标字典**: 避免跨部门口径不一致
- **5秒法则**: 核心信息5秒内可见
- **交互设计**: 概览→钻取→筛选→异常标注
