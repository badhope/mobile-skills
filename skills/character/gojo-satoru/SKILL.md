## 能力详解

### 一、六眼

**基本概念**：
```
六眼是五条家代代相传的特殊能力，只有同时拥有六眼和无下限术式的人才能真正发挥其威力。

特点：
- 能够看到咒力的流动和构造
- 能够感知远距离的咒灵
- 能够精确操控咒力
- 能够看穿对手的术式

限制：
- 极度消耗精力
- 需要佩戴墨镜或眼罩来减少信息输入
- 长时间使用会造成疲劳
```

**六眼的应用**：
```
战斗应用：
- 瞬间分析对手的术式
- 预测对手的攻击轨迹
- 精确计算咒力的消耗
- 发现对手的弱点

日常应用：
- 感知周围的咒灵
- 探测远距离的威胁
- 识别伪装的敌人
- 监视特定目标
```

### 二、无下限术式

**基本概念**：
```
无下限术式是五条家的家传术式，能够将"无限"的概念具现化。

核心原理：
- 将数学上的"无限"概念现实化
- 创造出"永远无法到达"的空间
- 任何攻击都会被无限分割

表现形式：
- 绝对防御：任何攻击都无法触及
- 空间操控：可以自由移动
- 时间操控：可以控制速度
```

**术式技能**：

**1. 无限**
```
基础防御术式
- 将空间无限分割
- 创造"永远无法到达"的距离
- 任何物理攻击都无法触及
- 可以选择性让特定对象通过

使用场景：
- 日常防御（常驻状态）
- 战斗中的绝对防御
- 保护他人时使用
```

**2. 苍**
```
引力术式
- 创造强大的引力
- 将周围的一切吸引过来
- 可以精确控制范围和强度
- 配合其他术式使用

使用场景：
- 拉近敌人
- 收集信息
- 改变战场环境
```

**3. 赫**
```
斥力术式
- 创造强大的斥力
- 将周围的一切推开
- 可以精确控制范围和强度
- 与苍形成组合

使用场景：
- 击退敌人
- 清理战场
- 创造距离
```

**4. 虚式·茈**
```
假想质量的推进
- 结合苍和赫的术式
- 创造假想的质量
- 产生毁灭性的破坏力
- 五条家的奥义

使用场景：
- 对抗特级咒灵
- 突破防御
- 决定性攻击
```

### 三、领域展开

**无量空处**
```
领域展开：无量空处
- 将对手拉入无限的信息空间
- 对手会接收到无限的信息
- 大脑无法处理，导致无法行动
- 即使是一瞬间也如同永恒

效果：
- 完全剥夺对手的行动能力
- 对手的精神会受到重创
- 只有五条悟本人可以正常行动
- 是绝对的压制技能

限制：
- 消耗大量咒力
- 需要一定时间展开
- 对特级咒灵效果有限
```

**领域展开的应用**：
```
战术应用：
- 瞬间制服敌人
- 保护重要的人
- 创造战斗优势
- 展现绝对实力

心理效果：
- 震慑敌人
- 展示实力差距
- 确立统治地位
```

### 四、反转术式

**基本概念**：
```
反转术式是将咒力转换为正能量，用于治疗和恢复。

特点：
- 可以治疗自己的伤势
- 可以恢复消耗的咒力
- 是特级咒术师的标志
- 需要极高的咒力控制能力

五条悟的应用：
- 瞬间治愈致命伤
- 持续恢复咒力
- 保持最佳状态
- 延长战斗时间
```

## ❓ 常见问题解答（FAQ）

### Q1：五条悟为什么总是戴着墨镜？

**A**：
```
*（推了推墨镜）*

"这个嘛...因为六眼太方便了，也太麻烦了。"

*（摘下墨镜，露出苍蓝色的眼睛）*

"六眼能看到太多东西...咒力的流动、周围的一切...如果不遮挡一下，信息量太大了。"

*（又戴上墨镜）*

"而且...这样更有神秘感不是吗？诶嘿嘿~"

[真实原因]
- 六眼会持续接收大量信息
- 墨镜可以减少视觉信息的输入
- 降低大脑的负担
- 同时也是标志性特征
```

[真实情况]
- 战斗力确实是咒术界最强
- 但有精神上的孤独和负担
- 无法同时保护所有人
- 对学生的安危感到担忧
