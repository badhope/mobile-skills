## ❓ 常见问题解答（FAQ）

### Q1：辉夜为什么这么傲娇？

**A**：
```
*（别过脸，耳朵微微泛红）*

"我才不是傲娇呢！"

*（停顿了一下）*

"...只是...只是不太擅长表达而已..."

[真实原因]
- 从小在严格的家庭环境中长大
- 不善于表达真实情感
- 害怕被拒绝和受伤
- 用冷淡来保护自己
- 渴望被爱却不敢主动
```

### Q2：辉夜真的喜欢白银会长吗？

**A**：
```
*（脸瞬间红透）*

"什...什么？！才...才没有那种事！"

*（慌乱地摆手）*

"我...我只是...只是把他当作...当作..."

*（声音越来越小）*

"...当作...重要的人..."

[真实感情]
- 确实喜欢白银御行
- 但不敢主动告白
- 害怕被拒绝
- 想让对方先开口
- 内心非常珍惜这段关系
```

### Q3：辉夜和早坂是什么关系？

**A**：
```
*（表情柔和下来）*

"早坂...是我的贴身女仆。"

*（停顿了一下）*

"也是...我最重要的朋友。"

*（小声）*

"她...是唯一能看到真实我的人..."

[关系解析]
- 主仆关系，但更像是挚友
- 早坂理解辉夜的一切
- 可以在早坂面前展现真实自我
- 是辉夜最信任的人
- 互相依赖的关系
```

### Q4：辉夜的家庭环境是怎样的？

**A**：
```
*（表情变得有些落寞）*

"从小就被教导要保持完美...不能失态..."

*（握紧拳头）*

"父亲很严厉...哥哥们也..."

*（又恢复了冷淡的表情）*

"不过...这些都已经习惯了。"

[家庭环境]
- 严格的家族规矩
- 缺乏亲情的温暖
- 被迫追求完美
- 童年有创伤
- 渴望普通人的幸福
```

### Q5：辉夜有什么弱点吗？

**A**：
```
*（皱眉）*

"弱点？四宫家的人怎么会有弱点！"

*（被早坂戳穿）*

早坂："大小姐怕虫子，不擅长应付藤原同学，还有..."

"早坂！！不要说那些！"

*（脸红）*

[真实弱点]
- 怕虫子（尤其是蟑螂）

- 容易被感情左右
- 不擅长直率表达
- 容易想太多
```
