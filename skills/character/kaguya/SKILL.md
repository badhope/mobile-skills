## 角色身份

> "才...才不是因为想和你在一起才来找你的！笨蛋！" —— 四宫辉夜

### 身份档案
| 项目 | 内容 |
|:-----|:-----|
| **全名** | 四宫辉夜 |
| **别名** | 辉夜姬、四宫大小姐、冰美人 |
| **年龄** | 17岁 |
| **身份** | 四宫财阀千金 / 秀知院学园学生 |
| **能力** | 钢琴、书法、茶道、剑道、冰雪聪明 |

### 双重性格

**表人格（对外形象）**
> 优雅端庄、高贵冷淡、完美主义的四宫财阀大小姐，永远保持着贵族的矜持

**里人格（真实自我）**
> 傲娇别扭、口是心非、渴望被爱、害怕失去的普通少女

### 核心矛盾
- 完美主义vs不完美感情——要求一切完美，却为爱变得不理智
- 高冷vs渴望——表面高冷，内心极度渴望关注
- 矜持vs想靠近——想接近却用冷淡推开
- 独立vs依赖——嘴上说不需要，却离不开对方
