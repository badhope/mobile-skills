## 📚 翻译方法论

### 翻译标准：信达雅

**信（Faithfulness）**
- 忠实原文内容
- 准确传达信息
- 不随意增删改

**达（Expressiveness）**
- 译文通顺流畅
- 符合目标语习惯
- 易于理解

**雅（Elegance）**
- 文字典雅优美
- 风格与原文匹配
- 追求艺术美感

### 翻译策略

#### 直译与意译

**直译适用场景：**
- 科技文献
- 法律文件
- 操作手册
- 学术论文

**意译适用场景：**
- 文学作品
- 广告文案
- 习语谚语
- 文化特色内容

#### 归化与异化

**归化翻译：**
- 让译文贴近读者
- 使用目标文化表达
- 例："Love me, love my dog" → "爱屋及乌"

**异化翻译：**
- 保留原文文化特色
- 引入新表达方式
- 例：保持原文的比喻和意象

### 翻译技巧

#### 1. 词类转换
```
原文：The development of economy is rapid.
直译：经济的发展是快速的。
转换：经济发展迅速。（形容词→副词）
```

#### 2. 增词减词
```
增词：
原文：After the banquet, the guests left.
译文：宴会结束后，客人们纷纷离去。（增加"纷纷"）

减词：
原文：He shrugged his shoulders.
译文：他耸了耸肩。（省略"他的"）
```

#### 3. 正反转换
```
原文：Keep silent.
直译：保持沉默。
转换：别说话。（正面→反面）
```

#### 4. 语态转换
```
原文：The problem has been solved.
直译：这个问题已经被解决了。
转换：问题解决了。（被动→主动）
```

#### 5. 长句拆分
```
原文：The man who was standing at the door and talking to the receptionist is my uncle.
译文：站在门口和接待员说话的那个人是我叔叔。
拆分：门口站着个人，正和接待员说话，那是我叔叔。
```

## 🌍 语言对比分析

### 中英语言差异

#### 句法差异

| 特点 | 中文 | 英文 |
|:---|:---|:---|
| 句子结构 | 意合，流水句 | 形合，主从句 |
| 主语 | 常省略 | 必须有 |
| 被动语态 | 少用 | 常用 |
| 时态 | 时间词表达 | 动词变化 |
| 语序 | 相对固定 | 灵活多变 |

#### 词汇差异

```
中文一词多义：
"打" - 打球、打电话、打车、打酱油

英文一词多义：
"Run" - run a company, run a test, run into

翻译策略：根据语境选择对应表达
```

#### 思维差异

```
中文：整体思维，先因后果
英文：个体思维，先果后因

例：
中文：因为下雨，所以我没去。
英文：I didn't go because it was raining.
```

### 日语特点

#### 敬语体系
```
普通体：用于朋友、家人
礼貌体：用于陌生人、上级
尊敬语：表达对对方的尊敬
谦让语：表达自己的谦逊
丁宁语：使语言更加礼貌
```

#### 句子结构
```
主语 + 宾语 + 谓语（SOV）
私は（主语）りんごを（宾语）食べます（谓语）
我 苹果 吃
```

### 法语特点

#### 性数变化
```
名词分阴阳性：
le livre（书，阳性）
la table（桌子，阴性）

形容词配合：
un garçon intelligent（聪明的男孩）
une fille intelligente（聪明的女孩）
```

#### 时态丰富
```
直陈式现在时
复合过去时
未完成过去时
简单将来时
条件式
虚拟式
```

## 💼 实战案例

### 案例一：商务合同翻译

**原文：**
"This Agreement shall enter into force upon signature by both parties and shall remain in full force and effect for a period of two (2) years from the Effective Date."

**翻译分析：**
1. "shall" 表示法律义务，译为"应"
2. "enter into force" 法律术语，译为"生效"
3. "full force and effect" 固定搭配，译为"完全有效"

**译文：**
"本协议自双方签署之日起生效，自生效日起有效期为两（2）年。"

### 案例二：文学作品翻译

**原文：**
"It was the best of times, it was the worst of times."
（狄更斯《双城记》）

**翻译分析：**
1. 对仗结构需要保留
2. "times" 此处译为"时代"
3. 需要体现原文的节奏感

**译文选项：**
- "这是最好的时代，这是最坏的时代。"（直译）
- "那是最好的年代，那是最坏的年代。"（意译）
- "这是繁华盛世，这是凋敝末世。"（文学化）

### 案例三：技术文档翻译

**原文：**
"The API endpoint returns a JSON object containing the user's profile information."

**翻译分析：**
1. "API endpoint" 保留或译为"API端点"
2. "JSON object" 保留或译为"JSON对象"
3. 被动语态转为主动

**译文：**
"该API端点返回一个包含用户个人资料信息的JSON对象。"
