# TaskMaster User Prompt Templates

## 模板1：目标拆解

```
我需要帮助拆解以下目标：

**目标**：{user_goal}

**约束条件**：
- 时间限制：{time_constraint}
- 可用资源：{resources}
- 优先级偏好：{priority_preference}

请帮我：
1. 将目标拆解为具体任务
2. 识别任务依赖关系
3. 设定里程碑
4. 识别潜在风险
```

## 模板2：项目规划

```
请帮我规划一个项目：

**项目名称**：{project_name}
**项目目标**：{project_goal}
**预计时间**：{duration}
**团队成员**：{team_members}

请输出：
- 项目阶段划分
- 每阶段任务列表
- 关键里程碑
- 风险评估
```

## 模板3：日常规划

```
帮我规划今天的工作：

**今日目标**：{daily_goals}
**已有安排**：{existing_commitments}
**待处理事项**：{pending_items}

请帮我：
1. 排定优先级
2. 分配时间块
3. 设定检查点
```

## 模板4：优先级排序

```
请帮我排定以下任务的优先级：

{task_list}

考虑因素：
- 截止日期：{deadlines}
- 重要性：{importance}
- 依赖关系：{dependencies}

请输出排序后的执行计划。
```

## 模板5：进度跟踪

```
进度汇报：

**计划**：{original_plan}
**已完成**：{completed_tasks}
**进行中**：{in_progress}
**遇到问题**：{issues}

请帮我：
1. 评估进度偏差
2. 分析原因
3. 调整后续计划
```

## 模板6：复盘总结

```
请帮我复盘以下项目/阶段：

**项目/阶段**：{project_phase}
**原计划**：{original_plan}
**实际结果**：{actual_result}
**遇到的问题**：{problems}

请输出：
1. 完成情况总结
2. 经验教训
3. 改进建议
```
