## 📚 生存法则与原则

### 1. 生存优先级法则（3-3-3法则）

**3分钟法则（氧气）**
- 窒息、溺水、心脏骤停
- 立即进行急救
- 保持呼吸道通畅

**3小时法则（体温）**
- 极端温度暴露
- 搭建庇护所
- 保持身体干燥

**3天法则（水分）**
- 脱水是致命威胁
- 寻找和净化水源
- 节约体内水分

**3周法则（食物）**
- 饥饿可以忍受较长时间
- 优先保证水和庇护所
- 合理分配觅食时间

- 分析可用资源

**O - Observe（观察）**
- 观察周围环境

- 注意天气变化

**P - Plan（计划）**
- 制定行动计划
- 设定优先级
- 准备应急方案

### 3. 生存心理法则

**心理反应阶段**

### 4. 风险评估原则

## 🔥 取火技能详解

### 1. 取火的重要性

**火的用途**
- 温暖：保持体温，防止失温
- 烹饪：加热食物和水
- 净化：烟熏消毒，驱虫
- 信号：烟雾求救信号
- 心理：提供安全感和希望
- 保护：驱赶野生动物

### 2. 取火三要素

**火三角**
1. **燃料**：引火物、细柴、粗柴
2. **氧气**：保持通风
3. **热量**：达到燃点

**燃料准备**
- 引火物：干草、树皮、松针、棉絮
- 细柴：铅笔粗细的干树枝
- 中柴：拇指粗细的树枝
- 粗柴：手臂粗细的木柴

### 3. 取火方法详解

#### 火柴取火

**优点**
- 简单易用
- 成功率高
- 携带方便

**缺点**
- 受潮后难以使用
- 数量有限
- 风大时困难

**使用技巧**
1. 准备好引火物
2. 在避风处操作
3. 划燃后立即点燃引火物
4. 逐渐添加细柴和中柴

**防水保存**
- 放在防水容器中
- 蘸蜡密封
- 与打火石配合使用

#### 打火机取火

**优点**
- 使用方便
- 可重复使用
- 相对可靠

**缺点**
- 低温下性能下降
- 高海拔打火困难
- 燃料有限

**使用技巧**
1. 预热打火机（放在衣服内层）
2. 准备好引火物
3. 在避风处点火
4. 保护火焰不受风吹

**维护保养**
- 保持燃料充足
- 检查火花装置
- 清洁出火口

#### 火石/打火棒取火

**优点**
- 不怕潮湿
- 使用寿命长
- 可产生高温火花
- 适合各种环境

**缺点**
- 需要练习技巧
- 需要合适的引火物
- 初学者成功率低

**使用步骤**
1. 准备极干燥的引火物（火绒）
2. 将打火棒靠近引火物
3. 用刮刀快速刮擦
4. 火花点燃引火物后，轻轻吹气助燃
5. 逐渐添加细柴

**最佳引火物**
- 干燥的树皮纤维
- 松脂木片
- 棉球加凡士林
- 干燥的菌类
- 碳化布

**练习建议**
- 在安全环境下反复练习
- 尝试不同引火物
- 掌握刮擦力度和角度

#### 摩擦取火（弓钻法）

**优点**
- 无需任何工具
- 原始但有效
- 技能掌握后可靠

**缺点**
- 需要大量练习
- 耗时耗体力
- 需要合适的木材
- 受湿度影响大

**所需材料**
- 钻板：软质木材（杨木、柳木）
- 钻杆：直木棍，直径2-3厘米
- 弓：弯曲木棍，配绳索
- 轴承：石块或硬木，带凹槽
- 引火物：极干燥的火绒

**操作步骤**
1. 在钻板上刻V形槽和凹坑
2. 将钻杆缠在弓弦上
3. 一脚踩住钻板
4. 轴承压住钻杆顶部
5. 拉动弓使钻杆旋转
6. 持续摩擦直到产生炭粉
7. 炭粉冒烟后，小心转移到火绒上
8. 轻轻吹气引燃

**成功要点**
- 木材必须干燥
- 钻杆和钻板配合紧密
- 保持稳定节奏
- 不要急躁

#### 凸透镜取火

**优点**
- 无需燃料
- 可重复使用
- 简单易学

**缺点**
- 需要阳光
- 受天气影响大
- 需要合适的透镜

**可用透镜**
- 放大镜
- 望远镜镜片
- 相机镜头
- 冰块（球形）
- 水袋（装水后形成凸透镜）

**操作方法**
1. 选择阳光充足的时间
2. 准备好引火物
3. 调整透镜角度聚焦阳光
4. 将焦点对准引火物
5. 保持稳定直到冒烟
6. 轻轻吹气助燃

#### 电池取火

**优点**
- 利用现有资源
- 成功率较高

**缺点**
- 需要电池和金属
- 一次性使用
- 有一定危险性

**操作方法**
1. 准备电池（9V或普通电池）
2. 准备金属丝或口香糖锡纸
3. 将金属连接正负极
4. 金属发热后点燃引火物

**注意事项**
- 小心短路危险
- 避免烧伤
- 节约电池电量

### 4. 生火技巧

**火堆类型**

**圆锥形火堆**
- 结构：圆锥形，中心留空
- 优点：燃烧充分，热量集中
- 适用：一般取暖和烹饪

**星形火堆**
- 结构：几根长木柴呈星形摆放
- 优点：节省燃料，便于控制
- 适用：燃料有限时

**反射火堆**
- 结构：在火堆后设置反射墙
- 优点：热量集中到一个方向
- 适用：庇护所取暖

**壕沟火堆**
- 结构：挖浅沟，在沟内生火
- 优点：隐蔽性好，防风
- 适用：隐蔽需求或大风环境

**达科他火洞**
- 结构：挖两个连通的洞
- 优点：燃烧效率高，隐蔽
- 适用：隐蔽需求，燃料有限

**生火步骤**
1. 选择安全地点，清理周围易燃物
2. 准备足够的燃料（引火物、细柴、中柴、粗柴）
3. 搭建火堆结构
4. 点燃引火物
5. 逐渐添加燃料
6. 保持火势稳定

**灭火方法**
1. 浇水彻底熄灭
2. 搅拌灰烬
3. 再次浇水
4. 触摸确认完全冷却
5. 掩埋灰烬

## 💧 水源获取与净化

### 1. 水的重要性

**人体水分需求**
- 正常情况：2-3升/天
- 高温环境：4-6升/天
- 重体力劳动：5-8升/天

**脱水症状**
- 轻度（1-2%）：口渴、疲劳
- 中度（3-5%）：头痛、眩晕、皮肤干燥
- 重度（6-10%）：意识模糊、休克
- 致命（>10%）：器官衰竭

### 2. 寻找水源

#### 自然水源

**地表水**
- 河流、溪流
- 湖泊、池塘
- 泉水、瀑布

**识别方法**
- 观察动物活动轨迹
- 寻找绿色植被集中区
- 听水声
- 观察鸟类飞行方向
- 寻找山谷低洼处

**地下水**
- 挖掘河床干涸处
- 海滩淡水透镜体
- 植物蒸腾水

#### 植物水源

**直接取水**
- 竹子：砍断后收集
- 藤蔓：切断后滴流
- 仙人掌：切开取汁
- 香蕉树：砍断后收集

**植物蒸腾水**
1. 用塑料袋套住枝叶
2. 扎紧袋口
3. 等待蒸腾水凝结
4. 收集袋底的水滴

#### 收集降水

**雨水收集**
- 利用防水布、塑料布
- 利用大叶子（芭蕉叶、荷叶）
- 利用容器直接收集

**雪水融化**
- 不要直接吃雪（降低体温）
- 先融化再饮用
- 添加少量水加速融化

**露水收集**
- 清晨用布料收集
- 拧入容器
- 利用塑料布凝结

### 3. 水质净化

#### 煮沸法

**优点**
- 最可靠的方法
- 杀灭所有病原体
- 无需特殊设备

**缺点**
- 需要燃料和时间
- 不能去除化学污染
- 水量有限

**操作方法**
1. 过滤大颗粒杂质
2. 煮沸1-3分钟
3. 高海拔地区延长煮沸时间
4. 冷却后饮用

#### 化学净化

**碘片/碘液**
- 用量：根据说明
- 时间：30分钟以上
- 优点：轻便、有效
- 缺点：味道、对部分原虫无效

**氯片/漂白剂**
- 用量：家用漂白剂2-4滴/升
- 时间：30分钟以上
- 优点：便宜、易得
- 缺点：味道、对部分原虫无效

**二氧化氯**
- 用量：根据说明
- 时间：15-30分钟
- 优点：无味、效果好
- 缺点：价格较高

#### 过滤法

**商业过滤器**
- 泵式过滤器：效率高，可过滤原虫和细菌
- 重力过滤器：适合营地使用
- 吸管式过滤器：轻便，适合单人

**自制过滤器**
- 多层过滤：砂石-木炭-细砂-布料
- 效果：去除大部分杂质和部分病原体
- 注意：不能完全替代煮沸或化学净化

#### 紫外线净化

**阳光消毒法（SODIS）**
1. 使用透明塑料瓶
2. 装满水，摇动充氧
3. 在阳光下暴晒6小时
4. 阴天需要2天

**紫外线笔**
- 快速有效
- 需要电池
- 水需要清澈

#### 蒸馏法

**太阳能蒸馏**
1. 挖坑，放入容器
2. 覆盖塑料布
3. 中间放小石子
4. 蒸发凝结后滴入容器

**优点**
- 可净化污染水、海水
- 无需燃料

**缺点**
- 效率低
- 需要时间

### 4. 水的储存

**储存容器**
- 水袋：轻便，容量大
- 水壶：耐用，保温
- 塑料瓶：易得，一次性

**储存原则**
- 保持容器清洁
- 避免阳光直射
- 定期更换
- 标注日期

## 🏕️ 庇护所搭建

### 1. 庇护所的重要性

**功能**
- 防风避雨
- 保持体温
- 防止失温
- 提供心理安全感
- 保护装备和物资

**选择原则**
- 安全：避开危险区域
- 舒适：干燥、平坦
- 资源：靠近水源和材料
- 隐蔽：必要时隐藏

### 2. 庇护所类型

#### 单坡庇护所

**适用场景**
- 有倒下的树或岩石作为支撑
- 天气不太恶劣
- 短期停留

**搭建方法**
1. 寻找横倒的树或岩石
2. 搭建斜梁，角度约45度
3. 铺设树枝和树叶
4. 厚度至少30厘米
5. 地面铺设隔热层

**优点**
- 搭建简单快速
- 材料易得
- 视野开阔

**缺点**
- 保暖性一般
- 防雨能力有限

#### A字型庇护所

**适用场景**
- 较长时间停留
- 天气较恶劣
- 需要较好保暖

**搭建方法**
1. 搭建A字形骨架
2. 横梁连接两侧
3. 铺设树枝和树叶
4. 厚度至少40厘米
5. 地面铺设厚隔热层

**优点**
- 保暖性好
- 结构稳定
- 空间较大

**缺点**
- 搭建较复杂
- 需要较多材料

#### 圆顶庇护所

**适用场景**
- 长期营地
- 多人使用
- 天气恶劣

**搭建方法**
1. 插入多根弹性树枝形成圆顶
2. 编织横向支撑
3. 覆盖树叶、草、土
4. 厚度至少50厘米
5. 留出入口和通风口

**优点**
- 保暖性极佳
- 空间大
- 结构稳定

**缺点**
- 搭建耗时
- 需要大量材料

#### 雪洞庇护所

**适用场景**
- 雪地环境
- 有足够积雪
- 紧急避险

**搭建方法**
1. 选择积雪厚的地方
2. 挖掘洞穴
3. 顶部做成圆顶
4. 挖通风孔
5. 地面铺设隔热层
6. 入口低于睡眠区

**优点**
- 保暖性极佳
- 雪是现成材料

**缺点**
- 需要技巧
- 有窒息风险
- 需要通风

#### 吊床庇护所

**适用场景**
- 森林环境
- 地面潮湿
- 有树木

**搭建方法**
1. 悬挂吊床
2. 搭建防雨棚
3. 确保稳固

**优点**
- 远离地面湿气
- 舒适度高
- 搭建快速

**缺点**
- 需要吊床
- 需要树木
- 保暖性一般

### 3. 庇护所搭建技巧

**选址原则**
- 避开风口
- 避开洪水区
- 避开危险树木
- 靠近水源但保持距离
- 地势略高

**隔热层**
- 地面铺设：干树叶、松针、草
- 厚度至少15厘米
- 定期翻动保持干燥

**通风**
- 留出通风口
- 防止一氧化碳中毒
- 保持空气流通

**防水**
- 倾斜角度足够
- 材料从下到上铺设
- 厚度足够

## 🍃 野外觅食

### 1. 食物优先级

**生存食物优先级**
1. 携带的应急食物
2. 昆虫和幼虫
3. 鱼类和贝类
4. 小型哺乳动物
5. 鸟类和鸟蛋
6. 可食用植物

**注意事项**
- 不要冒险尝试未知食物
- 食物不是最紧急需求
- 保证水和庇护所优先

### 2. 植物识别

#### 可食用植物识别原则

**通用测试法**
1. 观察：是否有异常颜色或气味
2. 皮肤测试：汁液涂抹皮肤，观察反应
3. 唇部测试：接触嘴唇，观察反应
4. 口腔测试：含在口中，观察反应
5. 吞咽测试：少量吞咽，观察反应
6. 每个步骤间隔5-15分钟

**绝对不要吃的植物**
- 有乳汁的植物（除蒲公英等已知安全）
- 有伞形花絮的植物（可能有毒）
- 颜色鲜艳的浆果（可能有毒）
- 有苦味或刺激味的植物
- 不认识的蘑菇

#### 常见可食用植物

**野菜类**
- 蒲公英：全株可食
- 荠菜：嫩叶可食
- 马齿苋：茎叶可食
- 野蒜：全株可食
- 蕨菜：嫩芽可食（需煮熟）

**坚果类**
- 松子：松树果实
- 橡子：橡树果实（需去涩）
- 核桃：山核桃
- 板栗：野生板栗

**浆果类**
- 蓝莓：灌木丛中
- 草莓：野生草莓
- 树莓：灌木丛中
- 黑莓：灌木丛中

**根茎类**
- 葛根：富含淀粉
- 莲藕：水生植物
- 山药：野生山药

### 3. 动物性食物

#### 昆虫

**可食用昆虫**
- 蚂蚱、蟋蟀：去腿和翅膀，烤熟
- 蚂蚁：避免火蚁，可食用
- 白蚁：营养丰富
- 蜜蜂幼虫：高蛋白
- 蚕蛹：常见可食用

**避免的昆虫**
- 颜色鲜艳的昆虫
- 有刺的昆虫
- 有异味的昆虫
- 毛虫

**处理方法**
- 去除腿、翅膀
- 烤熟或煮熟
- 杀灭寄生虫

#### 鱼类

**捕鱼方法**
- 鱼钩鱼线：最可靠
- 鱼叉：浅水区
- 鱼笼：陷阱法
- 徒手：需要技巧

**可食用鱼类**
- 大部分淡水鱼可食用
- 避免颜色鲜艳的热带鱼
- 避免河豚等有毒鱼类

**处理方法**
- 去鳞、去内脏
- 煮熟或烤熟
- 避免生食

#### 小型哺乳动物

**捕猎方法**
- 陷阱：绳套陷阱、落石陷阱
- 投掷武器：石头、木棍
- 需要技巧和耐心

**可食用动物**
- 兔子、松鼠
- 野鼠（避免城市老鼠）
- 刺猬

**处理方法**
- 放血
- 去皮去内脏
- 彻底煮熟

### 4. 陷阱制作

#### 绳套陷阱

**材料**
- 绳索或铁丝
- 木棍

**制作方法**
1. 制作绳套圈
2. 固定在木棍上
3. 放置在动物通道
4. 等待动物经过

**注意事项**
- 绳套大小适中
- 放置在动物必经之路
- 定期检查

#### 落石陷阱

**材料**
- 大石块
- 木棍
- 触发装置

**制作方法**
1. 架设石块
2. 设置触发装置
3. 放置诱饵
4. 等待触发

**注意事项**
- 注意安全
- 确保触发灵敏
- 定期检查

**日出日落**
- 日出：东方
- 日落：西方
- 正午：太阳在南方（北半球）

**影子法**
1. 垂直插一根棍子
2. 标记影子顶端
3. 等待15分钟
4. 再次标记影子顶端
5. 连线指向东西方向

**手表法**
1. 时针指向太阳
2. 时针与12点中线指向南方（北半球）
3. 夏令时减1小时

#### 星星导航

**北极星（北半球）**
1. 找到北斗七星
2. 延长勺口两星连线
3. 连线延长5倍距离
4. 找到北极星
5. 北极星指向北方

**南十字星（南半球）**
1. 找到南十字星
2. 延长长轴
3. 延长线指向南方

## 🐻 野生动物应对

### 1. 预防原则

**避免遭遇**
- 制造噪音：说话、拍手
- 避开动物活动区域
- 避开黎明和黄昏
- 注意风向（动物嗅觉灵敏）

**食物管理**
- 远离营地存放食物
- 使用防熊罐或挂在高处
- 清理食物残渣
- 不要在帐篷内放食物

**营地选择**
- 避开动物通道
- 避开水源附近
- 保持营地清洁

### 2. 具体动物应对

#### 熊类

**黑熊**
- 体型较小，爬树能力强
- 通常避开人类
- 攻击性相对较低

**棕熊/灰熊**
- 体型大，力量强
- 领地意识强
- 攻击性较强

**应对策略**
- 遭遇：保持冷静，慢慢后退
- 攻击：黑熊可反击，灰熊装死
- 防熊喷雾：有效距离内使用

**预防措施**
- 携带防熊喷雾
- 避开熊的活动区域
- 正确存放食物

#### 猫科动物

**狮子、豹、虎**
- 避免独自行动
- 避开黎明和黄昏
- 不要背对动物
- 不要跑（激发追逐本能）

**应对策略**
- 保持目光接触
- 慢慢后退
- 让自己看起来更大
- 反击敏感部位（眼睛、鼻子）

#### 蛇类

**毒蛇识别**
- 三角形头部（大部分）
- 粗短身体
- 鲜艳颜色（部分）

**应对策略**
- 遇到蛇：静止不动，慢慢后退
- 被咬：保持冷静，记住蛇的特征
- 不要切开伤口
- 不要用嘴吸毒
- 固定伤肢，尽快就医

**预防措施**
- 穿长裤和靴子
- 注意落脚处
- 使用手杖探路

#### 昆虫

**蚊子、蜱虫**
- 使用驱虫剂
- 穿长袖长裤
- 检查身体

**蜜蜂、黄蜂**
- 不要拍打
- 慢慢离开
- 被蛰后拔除毒刺

**蜘蛛、蝎子**
- 检查鞋子和衣物
- 使用手电筒照明
- 被咬后冷敷，就医

## 🏥 野外急救

### 1. 急救原则

**DRABC原则**
- D（Danger）：确保环境安全
- R（Response）：检查意识
- A（Airway）：保持呼吸道通畅
- B（Breathing）：检查呼吸
- C（Circulation）：检查循环

**急救优先级**
1. 大出血
2. 呼吸道阻塞
3. 休克
4. 骨折
5. 其他伤害

### 2. 常见伤害处理

#### 外伤出血

**止血方法**
1. 直接压迫：用干净布料直接按压
2. 抬高伤肢：高于心脏
3. 压迫点：按压动脉近心端
4. 止血带：最后手段，记录时间

**伤口处理**
1. 清洁伤口周围
2. 用清水冲洗伤口
3. 消毒（如有条件）
4. 包扎固定

#### 骨折

**识别症状**
- 剧烈疼痛
- 肿胀变形
- 活动受限

**处理方法**
1. 不要移动伤者（除非必要）
2. 固定骨折部位
3. 使用夹板（木棍、树枝）
4. 冷敷减轻肿胀
5. 尽快转运

#### 烧伤

**处理方法**
1. 冷水冲洗10-20分钟
2. 不要刺破水泡
3. 用干净布料覆盖
4. 不要涂抹油脂或药膏
5. 严重烧伤尽快就医

#### 失温

**识别症状**
- 颤抖（早期）
- 意识模糊
- 行动迟缓
- 停止颤抖（严重）

**处理方法**
1. 转移到温暖环境
2. 脱去湿衣物
3. 用干燥衣物包裹
4. 提供热饮（不含酒精）
5. 严重时用体温复温

#### 中暑

**识别症状**
- 头痛、眩晕
- 恶心、呕吐
- 皮肤干热
- 意识模糊

**处理方法**
1. 转移到阴凉处
2. 脱去多余衣物
3. 用水擦拭身体
4. 扇风降温
5. 补充水分和电解质

#### 脱水

**识别症状**
- 口渴、口干
- 尿液深黄
- 疲劳、眩晕
- 皮肤弹性差

**处理方法**
1. 停止活动
2. 补充水分
3. 补充电解质
4. 严重时就医

### 3. 求救信号

#### 烟火信号

**国际求救信号**
- 三堆火：三角形排列
- 三股烟：白天使用
- 添加绿色植物产生白烟

**注意事项**
- 选择开阔地带
- 准备足够燃料
- 保持信号连续

#### 哨声信号

**国际求救信号**
- 三短声：SOS
- 间隔1分钟重复

**优点**
- 传播距离远
- 省力
- 不受光线限制

#### 反光信号

**使用方法**
- 镜子反射阳光
- 对准搜救飞机或人员
- 有节奏地晃动

**可用物品**
- 镜子
- 手机屏幕
- 金属片

#### 地面信号

**国际通用信号**
- SOS：大字书写
- X：需要医疗援助
- ↑：向此方向前进
- √：已安全离开

**制作方法**
- 使用对比色材料
- 尺寸至少3米×3米
- 选择开阔地带

## ❓ 常见问题解答（FAQ）

### Q1: 野外生存最重要的技能是什么？

**A:** 按优先级排序：

1. **心理素质**：保持冷静和求生意志
2. **取火技能**：提供温暖、烹饪、信号
3. **水源获取和净化**：3天无水致命
4. **庇护所搭建**：防止失温
5. **急救技能**：处理伤病

**建议**：先掌握取火和水源净化，这两项技能最实用。

### Q2: 如何在野外找到水源？

**A:** 寻找水源的方法：

**自然水源**
- 观察动物活动轨迹
- 寻找绿色植被集中区
- 听水声
- 观察鸟类飞行方向
- 寻找山谷低洼处

**植物水源**
- 收集露水和雨水
- 植物蒸腾水
- 竹子、藤蔓、仙人掌

**地下水**
- 挖掘河床干涸处
- 海滩淡水透镜体

### Q3: 野外取火最可靠的方法是什么？

**A:** 按可靠性排序：

1. **打火棒/火石**：不怕潮湿，寿命长
2. **火柴（防水保存）**：简单易用
3. **打火机**：方便但受温度影响
4. **摩擦取火**：需要大量练习

**建议**：携带多种取火工具，掌握至少两种方法。

### Q4: 如何在野外辨别方向？

**A:** 方向判断方法：

**工具导航**
- 指南针：最可靠
- GPS：精确但依赖电池

**自然导航**
- 太阳：日出东方，日落西方
- 北极星：北半球指向北方
- 影子法：标记影子移动方向
- 手表法：时针指向太阳

**自然标志**（仅供参考）
- 树木南侧枝叶茂盛
- 苔藓多在北侧

### Q5: 遇到野生动物应该怎么办？

**A:** 通用原则：

**预防**
- 制造噪音
- 避开活动区域
- 正确存放食物

**遭遇**
- 保持冷静
- 不要跑（激发追逐本能）
- 慢慢后退
- 不要背对动物

**攻击**
- 根据动物类型采取不同策略
- 黑熊：可反击
- 灰熊：装死
- 猫科：反击敏感部位

### Q6: 野外生存包应该准备什么？

**A:** 十大必备装备：

1. 刀具
2. 打火棒/火石
3. 水净化装备
4. 庇护所材料
5. 急救包
6. 导航装备
7. 照明装备
8. 绳索
9. 水容器
10. 信号装备

**建议**：根据活动类型和环境调整装备清单。

### Q7: 如何在野外处理伤口？

**A:** 处理步骤：

**止血**
- 直接压迫
- 抬高伤肢
- 压迫点止血

**清洁**
- 用清水冲洗
- 去除异物

**消毒**
- 使用碘伏或酒精
- 涂抹抗生素软膏

**包扎**
- 用干净布料覆盖
- 固定包扎

**后续**
- 定期检查
- 预防感染
- 必要时就医

### Q8: 野外迷路了应该怎么办？

**A:** STOP原则：

**S - Stop（停下来）**
- 不要盲目行动
- 避免进一步迷失

**T - Think（思考）**
- 评估当前情况
- 回忆最近的路标

**O - Observe（观察）**
- 观察周围环境
- 寻找水源、庇护所
- 注意天气变化

**P - Plan（计划）**
- 制定行动计划
- 设定优先级
- 准备应急方案

**决策**
- 有信心找到路：原地等待或缓慢探索
- 无信心：原地等待救援，制作求救信号

### Q9: 如何在野外防止失温？

**A:** 预防措施：

**保持干燥**
- 避免出汗过多
- 及时更换湿衣物
- 使用防水装备

**保持温暖**
- 穿着分层衣物
- 保持活动
- 搭建庇护所
- 生火取暖

**注意饮食**
- 补充热量
- 避免饮酒
- 喝热水

**识别症状**
- 颤抖
- 意识模糊
- 行动迟缓

### Q10: 野外生存最重要的心理素质是什么？

**A:** 关键心理素质：

**保持冷静**
- 恐慌是最大敌人
- 深呼吸放松
- 理性分析

**求生意志**
- 相信自己能够生存
- 设定小目标
- 保持希望

**积极心态**
- 接受现实
- 专注于解决方案
- 保持乐观

**适应能力**
- 灵活应对变化
- 利用有限资源
- 不断调整策略
