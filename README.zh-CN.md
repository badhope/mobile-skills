| 步骤 | 操作 | 说明 |
|:---:|:---:|:---|
| 1️⃣ | **复制 Raw 链接** | 选择下方任意角色，复制其 Raw 链接 |

| 3️⃣ | **开始对话** | AI 将自动激活对应角色模式 |

```

```

## 🤝 贡献指南

查看 `docs/expansion-guide.md` 了解如何添加新角色。

**核心规则：**

| 规则 | 说明 |
|:---|:---|
| 1️⃣ 使用模板 | 始终使用适当的模板 |
| 2️⃣ 命名规范 | 遵循 kebab-case 命名约定 |
| 3️⃣ 双向更新 | 同时更新 README.md 和 README.zh-CN.md |
| 4️⃣ 必要章节 | 角色文件必须包含所有必要章节 |
| 5️⃣ 测试链接 | 提交前测试 Raw 链接 |
