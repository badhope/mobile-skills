# 资源经济系统

**文档版本**: 1.0
**核心设计理念**: 万物皆对象，万物有数值，万物可演化

---

## 系统概述

资源经济系统定义了九州仙途世界的货币流通、物品装备和生产技能体系，是支撑修仙世界运转的经济基础。

| 等级 | 名称 | 灵气含量 | 颜色 | 获取方式 |

```
1 仙石 = 100 极品灵石 = 10,000 上品灵石 = 1,000,000 中品灵石 = 100,000,000 下品灵石

1 极品灵石 = 100 上品灵石 = 10,000 中品灵石 = 1,000,000 下品灵石

1 上品灵石 = 100 中品灵石 = 10,000 下品灵石

1 中品灵石 = 100 下品灵石
```

#### 炼丹等级

| 等级 | 名称 | 可炼丹药 | 成功率加成 |
|:-----|:-----|:---------|:-----------|
| **入门** | 1级 | 一品丹药 | +0% |
| **小成** | 2级 | 二品丹药 | +10% |
| **大成** | 3级 | 三品丹药 | +20% |
| **圆满** | 4级 | 四品丹药 | +30% |
| **宗师** | 5级 | 五品丹药 | +40% |
| **丹圣** | 6级 | 六品丹药 | +50% |

#### 炼丹流程

```
1. 准备丹方
   └── 确认所需材料和丹炉

2. 准备材料
   └── 按丹方准备材料

3. 开始炼丹
   └── 进行炼丹检定

4. 炼丹检定
   └── 1d100 + 炼丹等级×10 + 悟性修正 vs 丹药难度

5. 判定品质
   ├── 大成功（>难度+30）：完美品质
   ├── 成功（>难度+15）：极品品质
   ├── 成功（>难度）：上品品质
   ├── 成功（≥难度）：中品品质
   ├── 失败（<难度-10）：下品品质
   └── 大失败（<难度-30）：炼丹失败，材料损毁

6. 完成炼丹
   └── 获得丹药
```

## 五、AI执行指南

### 5.1 物品生成流程

```
1. 确定物品类型
   └── 丹药/法宝/材料/符箓

2. 确定物品等级
   └── 根据产出来源确定

3. 进行品质检定
   └── 1d100 vs 品质表

4. 生成物品属性
   └── 根据等级和品质计算

5. 输出物品信息
   └── 格式化输出
```

### 5.2 交易判定流程

```
1. 确定交易物品
   └── 查询基础价格

2. 计算最终价格
   └── 基础价格 × 各项系数

3. 进行交易检定
   └── 1d20 + 修正 vs 10

4. 调整价格
   └── 根据检定结果调整

5. 完成交易
   └── 更新双方物品和货币
```

### 5.3 输出格式

#### 物品格式

```
【物品】[物品名称]（[品质]）
类型：[类型]
等级：[等级]
效果：[效果描述]
价格：[价格]
```

#### 交易格式

```
【交易】
物品：[物品名称]
原价：[原价]
🎲 交易检定：1d20 + [修正] = [结果]
最终价格：[最终价格]
```

### 5.4 示例

```
【物品】青云剑（上品）

价格：50上品灵石

【交易】
物品：青云剑（上品）
原价：50上品灵石
🎲 交易检定：1d20 + 2(声望) = 15 + 2 = 17
成功！价格优惠10%
最终价格：45上品灵石
```
