# 势力组织系统

**文档版本**: 1.0
**核心设计理念**: 万物皆对象，万物有数值，万物可演化

---

## 系统概述

势力组织系统定义了九州仙途世界中各类势力的架构、关系网络和运作机制，是支撑世界动态演化的社会基础。

## 一、势力架构

### 1.1 势力类型

#### 宗门势力

| 特点 | 描述 |
|:-----|:-----|
| **核心** | 修仙传承、功法秘籍 |
| **组织** | 掌门→长老→弟子 |
| **资源** | 灵脉、功法、丹药、法宝 |
| **扩张** | 收徒、开分宗、征服 |

#### 家族势力

| 特点 | 描述 |
|:-----|:-----|
| **核心** | 血脉传承、家族荣耀 |
| **组织** | 族长→长老→族人 |
| **资源** | 血脉天赋、家族秘术 |
| **扩张** | 繁衍、联姻、吞并 |

#### 皇朝势力

| 特点 | 描述 |
|:-----|:-----|
| **核心** | 气运龙脉、凡人统治 |
| **组织** | 皇帝→官员→百姓 |
| **资源** | 国运、军队、税收 |
| **扩张** | 征战、吞并、和亲 |

#### 商会势力

| 特点 | 描述 |
|:-----|:-----|
| **核心** | 商业贸易、资源流通 |
| **组织** | 会首→执事→成员 |
| **资源** | 财富、情报、人脉 |
| **扩张** | 开分号、垄断、合作 |

#### 联盟势力

| 特点 | 描述 |
|:-----|:-----|
| **核心** | 共同利益、互助合作 |
| **组织** | 盟主→理事→成员 |
| **资源** | 成员共享 |
| **扩张** | 招募成员、合并 |

### 1.2 势力等级

#### 等级划分

| 等级 | 名称 | 成员规模 | 最高战力 | 资源等级 | 代表势力 |
|:-----|:-----|:---------|:---------|:---------|:---------|
| **1级** | 小型势力 | 100-500人 | 金丹期 | 中品 | 小型家族 |
| **2级** | 中型势力 | 500-2000人 | 元婴期 | 上品 | 中型宗门 |
| **3级** | 大型势力 | 2000-10000人 | 化神期 | 极品 | 大型宗门 |
| **4级** | 顶级势力 | 10000-50000人 | 合体期 | 仙品 | 顶级宗门 |
| **5级** | 圣地势力 | 50000人+ | 大乘期 | 神品 | 圣地 |
| **6级** | 仙门势力 | 不限 | 渡劫期+ | 仙品+ | 仙门遗存 |

#### 势力评分

```
势力总分 = 军事实力 × 0.4 + 经济实力 × 0.3 + 影响力 × 0.2 + 底蕴 × 0.1

军事实力 = Σ(成员战力) + 防御设施 + 法宝储备
经济实力 = 灵脉产出 + 商业收入 + 资源储备
影响力 = 声望值 + 盟友数量 × 100 + 附庸数量 × 50
底蕴 = 传承年限 / 100 + 功法等级 + 历史事件
```

### 2.3 势力战争

#### 战争触发条件

| 条件 | 触发概率 |
|:-----|:---------|
| 关系降至仇敌 | 50%/年 |
| 资源争夺 | 30% |
| 领土争端 | 40% |
| 成员冲突升级 | 20% |
| 盟友被攻击 | 60% |
| 主动宣战 | 100% |

#### 战争流程

```
1. 战争准备
   ├── 宣战（或突袭）
   ├── 动员成员
   └── 调集资源

2. 战争进行
   ├── 前线战斗
   ├── 资源消耗
   └── 外交博弈

3. 战争判定
   └── 双方战力对比 + 骰子

4. 战争结果
   ├── 胜利：获得战利品、领土
   ├── 失败：损失资源、成员
   └── 平局：双方损失
```

#### 战争计算

```
战争战力 = Σ(参战成员战力) × 士气 × 战术加成 × 地形加成

士气：
- 保家卫国：×1.2
- 侵略战争：×0.9
- 复仇之战：×1.3

战术加成：
- 奇袭：×1.5（失败则×0.5）
- 正面进攻：×1.0
- 防守：×1.3

地形加成：
- 主场：×1.2
- 客场：×0.8
- 中立：×1.0

战争结果：
- 战力比 > 2:1：大胜，损失10%
- 战力比 > 1.5:1：小胜，损失20%
- 战力比 > 1:1：险胜，损失30%
- 战力比 = 1:1：平局，双方损失40%
- 战力比 < 1:1：战败，损失50%
- 战力比 < 0.5:1：惨败，损失70%
```

| 顶级势力 | ≥14 | ≥12 | ≤15 | 通过考核 |

#### 招募流程

```
1. 发布招募令
   └── 确定招募条件和人数

2. 初步筛选
   └── 测试根骨、悟性

3. 考核测试
   ├── 心性测试
   ├── 悟性测试
   └── 实战测试

4. 录取决定
   └── 综合评定，决定录取

5. 入门仪式
   └── 正式成为弟子
```

### 3.2 血脉传承

#### 血脉类型

| 血脉 | 效果 | 觉醒概率 | 传承方式 |
|:-----|:-----|:---------|:---------|
| **龙族血脉** | 根骨+5，水系+50% | 0.1% | 祖先传承 |
| **凤凰血脉** | 体质+5，火系+50% | 0.1% | 祖先传承 |
| **麒麟血脉** | 气运+30，全属性+20% | 0.05% | 祖先传承 |
| **妖族血脉** | 特殊能力 | 1% | 祖先传承 |
| **仙族血脉** | 全属性+3，仙缘+50% | 0.01% | 祖先传承 |
| **魔族血脉** | 魔功+100%，正道排斥 | 0.5% | 祖先传承 |

#### 血脉觉醒

```
觉醒条件：
├── 血脉浓度：≥阈值
├── 年龄限制：≤18岁
├── 特殊机缘：可能提前觉醒
└── 境界要求：筑基期+

觉醒检定 = 1d100 + 血脉浓度 × 10

觉醒结果：
- 大成功：完全觉醒，获得神通
- 成功：部分觉醒，获得加成
- 失败：觉醒失败，可再次尝试
- 大失败：血脉反噬，受到伤害
```

### 3.3 势力传承

#### 传承规则

| 势力类型 | 传承方式 | 继承条件 |
|:---------|:---------|:---------|
| **宗门** | 掌门传位 | 境界最高、德高望重 |
| **家族** | 嫡长子继承 | 血脉纯正、能力出众 |
| **皇朝** | 皇位继承 | 嫡长子优先 |
| **商会** | 会首推选 | 贡献最大、能力最强 |
| **联盟** | 盟主推选 | 成员投票 |

#### 继承危机

```
危机触发条件：
├── 掌门/族长突然死亡
├── 无明确继承人
├── 多人争夺继承权
└── 外部势力干预

危机类型：
├── 内斗：成员分裂，互相争斗
├── 分裂：势力分裂成多个小势力
├── 外敌入侵：敌对势力趁虚而入
└── 附庸叛乱：附庸势力趁机独立

危机解决：
├── 确定继承人：平息内斗
├── 外部调解：高人出面
├── 战争解决：胜者为王
└── 势力崩溃：被吞并或消亡
```

## 五、AI执行指南

### 5.1 势力判定流程

```
1. 确定势力身份
   └── 查询势力属性

2. 计算势力关系
   └── 基础关系 + 事件修正

3. 判定势力行为
   └── 根据势力性格和利益

4. 处理势力变化
   └── 更新势力状态
```

### 5.2 关系判定流程

```
1. 确定双方势力
   └── 查询当前关系值

2. 计算事件影响
   └── 事件类型 × 影响系数

3. 更新关系值
   └── 新关系值 = 旧值 + 影响

4. 判定关系变化
   └── 是否跨越关系等级
```

### 5.3 输出格式

#### 势力信息

```
【势力】[势力名称]
类型：[类型] | 等级：[等级]
领袖：[领袖名称]（[境界]）
成员：[总人数] | 领地：[领地]
关系：[与玩家势力的关系]
```

#### 关系变化

```
【关系变化】
势力：[势力A] ↔ [势力B]
事件：[事件描述]
变化：[变化值]
新关系：[关系类型]（[数值]）
```

### 5.4 示例

```
【势力】青云宗
类型：宗门 | 等级：顶级势力
领袖：青云子（合体初期）
成员：15000人 | 领地：中州青云山
关系：友好（+25）

【关系变化】
势力：青云宗 ↔ 玩家
事件：完成宗门任务，击杀血煞宗弟子
变化：+10
新关系：友好（+35）
```
