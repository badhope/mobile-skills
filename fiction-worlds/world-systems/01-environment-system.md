# 天地环境系统

**文档版本**: 1.0
**核心设计理念**: 万物皆对象，万物有数值，万物可演化

## 系统概述

天地环境系统是九州仙途世界的物理引擎，决定了修仙的"硬条件"和"地图格局"。该系统包含地理空间架构、灵气系统、天道法则系统和特殊地域四大模块。

#### 灵脉演化规则

```
灵脉状态变化：
├── 自然演化
│   ├── 生长：每百年 +1~5% 健康（有灵眼滋养）
│   ├── 衰退：每百年 -1~3% 健康（无灵眼或过度开采）
│   └── 消亡：健康值归零，灵脉枯竭
│
├── 人为影响
│   ├── 开采：每次开采 -0.1~1% 健康
│   ├── 战争：战斗破坏 -5~20% 健康
│   ├── 培育：阵法培育 +0.5~2% 健康/年
│   └── 夺取：势力更替，所有权变更
│
└── 特殊事件
    ├── 地震：-10~30% 健康
    ├── 天劫：可能强化或破坏
    └── 仙缘：可能提升等级
```

#### 浓度计算

```
区域灵气浓度 = 基础浓度 + 灵脉加成 + 灵眼加成 - 消耗减益 + 季节修正

基础浓度：根据区域类型确定
灵脉加成：灵脉等级 × 5
灵眼加成：灵眼类型加成（聚灵眼+50%）
消耗减成：修炼者数量 × 0.01
季节修正：春季+5，夏季+0，秋季-5，冬季-10
```

#### 灵脉供给

```
灵脉供给量 = 灵脉产出 × (1 - 距离/覆盖范围) × 健康度/100

示例：
青云灵脉（巨型灵脉，产出5000/时辰，覆盖100公里）
距离50公里处：5000 × (1 - 50/100) × 0.85 = 2125/时辰
距离90公里处：5000 × (1 - 90/100) × 0.85 = 425/时辰
```

## 五、环境演化规则

### 5.1 自然演化

```
时间尺度：以年为单位

灵脉演化：
├── 生长：每百年 +1~5% 健康（有灵眼滋养）
├── 衰退：每百年 -1~3% 健康（无灵眼或过度开采）
└── 消亡：健康值归零，灵脉枯竭

区域变化：
├── 灵气浓度：随灵脉状态变化
├── 植被生长：受灵气浓度影响
└── 妖兽分布：受灵气浓度和资源影响

天道变化：
├── 世界等级：每万年可能升降
├── 法则完整度：随世界状态变化
└── 天道意志：随世界事件变化
```

### 5.2 人为影响

```
灵脉开采：
├── 温和开采：-0.1% 健康/年
├── 正常开采：-0.5% 健康/年
├── 过度开采：-2% 健康/年
└── 涸泽而渔：-10% 健康/年

战争破坏：
├── 小型冲突：-1~5% 健康
├── 大型战争：-10~30% 健康
└── 灭门之战：-30~50% 健康

培育恢复：
├── 基础阵法：+0.5% 健康/年
├── 高级阵法：+1% 健康/年
├── 仙级阵法：+2% 健康/年
└── 神级阵法：+5% 健康/年
```

### 5.3 特殊事件

| 事件类型 | 触发条件 | 影响 |
|:---------|:---------|:-----|
| **天地大劫** | 世界周期 | 世界等级下降，灵脉受损 |
| **灵气复苏** | 特殊机缘 | 灵气浓度大幅提升 |
| **天降异宝** | 天道恩赐 | 区域灵脉等级提升 |
| **妖兽潮** | 妖兽周期 | 区域危险等级上升 |
| **仙门开启** | 特殊条件 | 仙界灵气涌入 |

## 六、AI执行指南

### 6.1 环境判定流程

```
1. 确定玩家位置
   └── 查询区域属性

2. 计算灵气浓度
   └── 基础浓度 + 灵脉加成 + 灵眼加成 - 消耗减益

3. 判定环境影响
   ├── 修炼效率修正
   ├── 战斗环境影响
   └── 特殊地域效果

4. 处理环境变化
   ├── 记录消耗
   ├── 更新状态
   └── 触发演化
```

### 6.2 输出格式

```
【环境信息】
📍 位置：[区域名称]
🌡️ 灵气浓度：[数值] ([等级])
⚡ 灵脉状态：[健康度]%
⚠️ 危险等级：[等级]
🌀 特殊效果：[效果列表]

[环境描述]
```

【环境信息】
