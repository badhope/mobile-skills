# 因果事件系统

**文档版本**: 1.0
**核心设计理念**: 万物皆对象，万物有数值，万物可演化

---

## 系统概述

因果事件系统是九州仙途世界的动态引擎，通过随机事件池、因果链条和事件演化机制，驱动世界不断变化和发展。

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           因果事件系统架构                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   随机事件池                                                                 │
│   ├── 天灾事件（自然灾害、灵气异变）                                         │
│   ├── 人祸事件（冲突、战争、阴谋）                                           │
│   ├── 机缘事件（奇遇、宝物、传承）                                           │
│   └── 日常事件（生活、修炼、社交）                                           │
│                                                                             │
│   因果链条                                                                   │
│   ├── 仇恨系统（仇恨值、复仇事件）                                           │
│   ├── 气运干涉（气运影响、机缘分配）                                         │
│   └── 因果清算（因果报应、天劫影响）                                         │
│                                                                             │
│   事件演化                                                                   │
│   ├── 事件触发（条件检测、概率判定）                                         │
│   ├── 事件链（连锁反应、蝴蝶效应）                                           │
│   └── 世界演化（势力变化、格局变迁）                                         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 二、因果链条

### 2.1 仇恨系统

#### 仇恨值计算

| 行为 | 仇恨值变化 | 持续时间 |
|:-----|:-----------|:---------|
| **击杀族人** | +50~100 | 永久 |
| **重伤族人** | +20~50 | 10年 |
| **抢夺资源** | +10~30 | 5年 |
| **侮辱挑衅** | +5~20 | 1年 |
| **救助仇敌** | +10~30 | 5年 |
| **化解恩怨** | -20~50 | 永久 |

#### 仇恨等级

| 等级 | 仇恨值 | 效果 |
|:-----|:-------|:-----|
| **无** | 0 | 正常 |
| **轻微** | 1-20 | 偶有冲突 |
| **中等** | 21-50 | 主动挑衅 |
| **严重** | 51-80 | 追杀 |
| **不死不休** | 81-100 | 全面追杀 |

#### 复仇事件

```
复仇触发条件：
├── 仇恨值 ≥ 50
├── 实力差距 ≤ 2个大境界
└── 距离 ≤ 1000公里

复仇方式：
├── 亲自追杀
├── 悬赏追杀
├── 派遣弟子
└── 联合仇家

复仇结果：
├── 成功：仇恨值清零，可能获得战利品
├── 失败：仇恨值+10，可能被反杀
└── 和解：仇恨值-30，可能触发后续事件
```

## 三、事件演化

### 3.1 事件触发

#### 触发条件检测

```
事件触发检测流程：
1. 检测时间触发条件
   └── 是否到达特定时间

2. 检测地点触发条件
   └── 是否到达特定地点

3. 检测状态触发条件
   └── 是否满足状态条件

4. 检测概率触发条件
   └── 进行概率检定

5. 综合判定
   └── 所有条件满足则触发
```

#### 触发概率计算

```
最终概率 = 基础概率 × 气运修正 × 环境修正 × 状态修正

气运修正：
- 气运 1-20：×0.5
- 气运 21-40：×0.8
- 气运 41-60：×1.0
- 气运 61-80：×1.2
- 气运 81-100：×1.5

环境修正：
- 灵气浓郁：×1.2
- 灵气稀薄：×0.8
- 特殊地域：×1.5~2.0

状态修正：
- 好状态：×1.2
- 坏状态：×0.8
- 正常：×1.0
```

### 3.2 事件链

#### 连锁事件

```
事件链示例：

【事件链：青云宗危机】
├── 触发：青云宗灵脉受损
│   ├── 分支A：修复灵脉
│   │   └── 后续：青云宗恢复，好感+20
│   └── 分支B：坐视不管
│       └── 后续：青云宗衰落，好感-20
│           └── 后续：血煞宗趁机进攻
│               ├── 分支A：协助防御
│               │   └── 后续：青云宗感激，结盟
│               └── 分支B：袖手旁观
│                   └── 后续：青云宗覆灭，正道势力削弱
```

#### 蝴蝶效应

```
蝴蝶效应规则：
├── 小事件可能引发大变化
├── 玩家行为会影响世界走向
├── 势力格局会动态变化
└── 历史会因玩家而改变

示例：
玩家救了一个凡人小孩
→ 小孩长大后成为天才修士
→ 成为某宗门长老
→ 在势力战争中起关键作用
→ 改变势力格局
```

## 五、AI执行指南

### 5.1 事件触发流程

```
1. 检测触发条件
   ├── 时间检测
   ├── 地点检测
   ├── 状态检测
   └── 概率检测

2. 计算触发概率
   └── 基础概率 × 各项修正

3. 进行概率检定
   └── 1d100 ≤ 触发概率

4. 触发事件
   └── 输出事件描述和选项

5. 处理玩家选择
   └── 执行对应结果
```

### 5.2 事件链处理流程

```
1. 检测事件链触发
   └── 特定事件发生时

2. 初始化事件链
   └── 记录事件链状态

3. 处理当前节点
   └── 执行节点事件

4. 检测分支条件
   └── 确定下一节点

5. 更新事件链状态
   └── 进入下一节点或结束
```

### 5.3 输出格式

#### 事件触发

```
【事件触发】[事件名称]
[事件描述]

可选行动：
1. [选项1]
2. [选项2]
3. [选项3]

请选择你的行动：
```

#### 事件结果

```
【事件结果】
选择：[玩家选择]
🎲 [检定类型]：[结果]
[结果描述]

[状态变化]
```
