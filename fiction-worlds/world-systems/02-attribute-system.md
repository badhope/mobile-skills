# 修真属性体系

**文档版本**: 1.0
**核心设计理念**: 万物皆对象，万物有数值，万物可演化

---

## 系统概述

修真属性体系定义了九州仙途中所有生命体的核心属性框架，包括生命状态板、修为境界体系和功法技能树三大模块。

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           修真属性体系架构                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   生命状态板                                                                 │
│   ├── 先天属性（根骨、悟性、体质、神识、气运）                               │
│   ├── 后天属性（寿元、灵力、体力、心魔）                                     │
│   └── 隐藏属性（因果、天道好感、命格）                                       │
│                                                                             │
│   修为境界体系                                                               │
│   ├── 凡人境界（炼气→筑基→金丹→元婴→化神→合体→大乘→渡劫）                   │
│   ├── 境界突破（条件、概率、风险）                                           │
│   └── 战力评估（境界战力、越级战斗）                                         │
│                                                                             │
│   功法技能树                                                                 │
│   ├── 功法体系（功法等级、功法类型）                                         │
│   ├── 术法体系（术法等级、术法类型）                                         │
│   └── 神通体系（神通等级、神通类型）                                         │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

| **气运** | QY | 运气造化 | 1-100 | 随机事件、机缘概率 |

#### 属性等级

| 等级 | 数值 | 描述 | 出现概率 |
|:-----|:-----|:-----|:---------|
| **极差** | 1-4 | 天生缺陷 | 5% |
| **平庸** | 5-8 | 凡人水平 | 60% |
| **优秀** | 9-12 | 天赋出众 | 25% |
| **卓越** | 13-16 | 万中无一 | 8% |
| **绝世** | 17-20 | 千年难遇 | 1.9% |
| **传说** | 21+ | 神话级别 | 0.1% |

#### 属性修正

```
属性修正 = floor((属性值 - 10) / 2)

示例：
- 属性 5 → 修正 -2
- 属性 8 → 修正 -1
- 属性 10 → 修正 0
- 属性 12 → 修正 +1
- 属性 14 → 修正 +2
- 属性 17 → 修正 +3
- 属性 20 → 修正 +5
```

#### 属性生成规则

```
初始属性分配：
├── 总属性点：72点
├── 单项范围：5-18
├── 分配方式：
│   ├── 自由分配
│   ├── 随机生成（4d6取3，6次）
│   └── 购点制（每点消耗1，超过15每点消耗2）
│
└── 气运单独计算：
    ├── 基础气运：1d100
    ├── 可通过因果改变
    └── 上限100，下限1
```

#### 突破检定

```
突破检定 = 1d100 + 根骨修正 + 功法加成 + 丹药加成 + 环境加成 + 因果修正

难度：
- 炼气→筑基：50
- 筑基→金丹：60
- 金丹→元婴：70
- 元婴→化神：80
- 化神→合体：85
- 合体→大乘：90
- 大乘→渡劫：95

加成来源：
├── 根骨修正：根骨 × 2
├── 功法加成：功法等级 × 5
├── 丹药加成：丹药品质 × 10
├── 环境加成：灵气浓度 / 10
└── 因果修正：因果值 / 10

结果：
├── 大成功（>难度+20）：突破成功，获得额外奖励
├── 成功（≥难度）：突破成功
├── 失败（<难度）：突破失败，损失资源
└── 大失败（<难度-20）：突破失败，重伤/境界下降
```

#### 战力计算

```
战力 = 基础战力 × (1 + 属性修正) × (1 + 功法加成) × (1 + 法宝加成) × (1 + 状态加成)

属性修正 = (根骨+体质+神识) / 30
功法加成 = 功法等级 × 0.1
法宝加成 = 法宝等级 × 0.2
状态加成 = 状态效果（狂暴+0.5，虚弱-0.3等）
```

## 五、AI执行指南

### 5.1 属性判定流程

```
1. 确定检定类型
   └── 选择相关属性

2. 计算修正值
   └── 属性修正 + 技能等级 + 其他加成

3. 进行检定
   └── 1d20 + 修正值 vs 难度

4. 判定结果
   ├── 大成功：效果翻倍
   ├── 成功：正常效果
   ├── 失败：无效果/负面效果
   └── 大失败：额外负面后果
```

### 5.2 境界突破流程

```
1. 检查前置条件
   ├── 境界是否圆满
   ├── 资源是否充足
   └── 状态是否良好

2. 计算成功率
   └── 基础概率 + 各项加成

3. 进行突破检定
   └── 1d100 + 加成 vs 难度

4. 处理结果
   ├── 成功：境界提升
   ├── 失败：资源损失
   └── 大失败：额外惩罚
```

### 5.3 输出格式

```
【检定】[检定类型]
🎲 1d20 + [修正] = [结果] vs [难度]
[成功/失败]！[结果描述]
```

### 5.4 示例

```
【检定】修炼检定
🎲 1d20 + 2(根骨) + 3(功法) = 15 + 5 = 20 vs 15
成功！今日修炼进展顺利，修为+3。

【检定】突破检定（筑基→金丹）
🎲 1d100 + 28(根骨) + 20(功法) + 30(丹药) = 45 + 78 = 123 vs 60
大成功！金丹凝结成功，品质上佳！
```
