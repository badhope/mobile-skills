## 1. 世界概述

### 1.1 世界简介

这是一个名为「九州」的修仙世界。天地间灵气充盈，万物皆可修炼成精。凡人通过吸收天地灵气，淬炼肉身神魂，可踏上修仙之路，追求长生不老、超脱轮回。

玩家将扮演一名初入修仙界的凡人，从最底层开始，经历修炼、历练、机缘、劫难，最终追求那虚无缥缈的大道。

### 1.2 核心主题

| 主题 | 描述 |
|:-----|:-----|
| **逆天改命** | 凡人亦可逆天，命运掌握在自己手中 |

| **机缘造化** | 危机与机遇并存，一念天堂一念地狱 |
| **因果循环** | 种善因得善果，善恶终有报 |

## 2. 世界观设定

### 2.1 历史背景

#### 远古时代

天地初开，混沌分化，诞生了第一批生灵——先天神魔。他们天生拥有大神通，掌控天地法则。神魔大战持续万载，最终以双方同归于尽告终。神魔陨落后，其精血、残魂散落天地，化作无数灵脉、秘境、传承。

#### 上古时代

人族崛起，有先贤参悟天地，创立修仙之法。各大修仙宗门相继建立，人族进入修仙盛世。然而，一场名为「天地大劫」的灾难降临，无数宗门覆灭，修仙界元气大伤。

#### 当今时代

距离天地大劫已过三万年。修仙界逐渐恢复，但再无上古时期的辉煌。各大宗门割据一方，正邪势力明争暗斗。而传说中的「飞升之道」，已无人能够触及。

## 4. 游戏机制

### 4.1 骰子/随机系统

#### 基础检定

```
检定公式：1d20 + 属性修正 + 技能等级 + 其他加成

大成功：骰出 20
- 结果自动成功
- 额外获得正面效果

大失败：骰出 1
- 结果自动失败
- 产生额外负面后果
```

#### 战斗检定

```
攻击检定：1d20 + 攻击属性修正 + 武器加成
防御检定：10 + 防御属性修正 + 防具加成

伤害计算：
- 物理伤害 = 武器伤害 + 力量修正
- 法术伤害 = 技能基础伤害 + 灵力修正
- 最终伤害 = 伤害 - 防御值
```

#### 修炼检定

```
修炼进度 = 1d20 + 根骨修正 + 功法加成 + 环境加成

修炼结果：
- 1-5：修炼受阻，进度+0
- 6-10：进展缓慢，进度+1
- 11-15：正常修炼，进度+2
- 16-19：进展顺利，进度+3
- 20：顿悟！进度+5，获得额外感悟
```
