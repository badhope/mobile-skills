# AI Agent Navigation System

> 通过 Raw 链接激活 AI 角色——跨平台、无门槛、即刻使用

---

## 如何使用（3 步完成）

| 步骤 | 操作 | 说明 |
|:---:|:---:|:---|
| 1 | 复制角色 Raw 链接 | 选择下方任意角色，复制其 Raw 链接 |
| 2 | 发送给 AI | 将链接发送给任何支持联网的 AI |
| 3 | 开始对话 | AI 将自动激活对应角色模式 |

**推荐激活语**：
```
请读取以下文件并切换到对应角色模式：
[Raw 链接]
```

## 扩展指南

### 添加新角色

1. 基于 `templates/role-template.md` 创建新文件
2. 按分类放入对应目录（如 `agents/companion/`）
3. 在本 README 添加角色信息
4. 更新仓库结构文档

### 角色命名规范

- 文件名：`kebab-case`（如 `product-strat.md`）
- 角色 ID：英文小写（如 `coder`、`kaguya`）
- 目录名：`kebab-case`（如 `agents/companion/`）
